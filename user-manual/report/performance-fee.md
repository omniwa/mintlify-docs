# Performance Fee

### Performance Fee

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (252).png" alt=""><figcaption></figcaption></figure></div>

| **Component** | **Description (English)**                               | **คำอธิบาย (ภาษาไทย)**                                        |
| ------------- | ------------------------------------------------------- | ------------------------------------------------------------- |
| Year          | The specific reporting year for the performance fee.    | ปีปฏิทินที่ต้องการบันทึกหรือเรียกดูค่าธรรมเนียมผลการดำเนินงาน |
| Portfolio     | The specific portfolio or fund being assessed for fees. | ชื่อพอร์ตโฟลิโอหรือกองทุนที่ต้องการจัดการข้อมูลค่าธรรมเนียม   |
| Create Button | Open a form to create a new performance fee entry.      | ปุ่มสำหรับสร้างรายการบันทึกค่าธรรมเนียมผลการดำเนินงานใหม่     |
| Data Table    | Displays recorded performance fee details and history.  | ตารางแสดงรายละเอียดประวัติการบันทึกค่าธรรมเนียมที่ผ่านมา      |
| Edit Button   | Modify an existing performance fee record.              | ปุ่มสำหรับแก้ไขข้อมูลรายการค่าธรรมเนียมที่บันทึกไว้แล้ว       |



### Import tab

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (253).png" alt=""><figcaption></figcaption></figure></div>

| **Component**       | **Description (English)**                                   | **คำอธิบาย (ภาษาไทย)**                                         |
| ------------------- | ----------------------------------------------------------- | -------------------------------------------------------------- |
| Select Date         | Select the specific date or date range for the calculation. | เลือกวันที่หรือช่วงเวลาที่ต้องการประมวลผลข้อมูล                |
| Download Template   | Download the standardized Excel template for data entry.    | ปุ่มสำหรับดาวน์โหลดไฟล์แม่แบบ Excel มาตรฐานของระบบ             |
| Upload Button       | Submit the completed Excel file into the system.            | ปุ่มสำหรับเลือกและอัปโหลดไฟล์ที่กรอกข้อมูลเรียบร้อยแล้ว        |
| Data Table          | Displays uploaded records that are waiting for review.      | ตารางแสดงรายการข้อมูลที่อัปโหลดขึ้นมาและกำลังรอการตรวจสอบ      |
| Submit Valid Record | Process and finalize entries that have passed validation.   | ปุ่มสำหรับยืนยันการนำข้อมูลที่ตรวจสอบแล้วว่าถูกต้องเข้าสู่ระบบ |
