# Performance Fund Policy

### Performance Fund Policy

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (55).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                                | **คำอธิบาย (ภาษาไทย)**                                |
| --------------------- | -------------------------------------------------------- | ----------------------------------------------------- |
| Select Policy         | Choose the specific fund policy to evaluate.             | เลือกนโยบายกองทุนที่ต้องการประเมินผล                  |
| Select Date           | Select the specific reporting date or period.            | เลือกวันที่หรือช่วงเวลาที่ต้องการเรียกดูรายงาน        |
| Search Button         | Execute the search based on selected criteria.           | ปุ่มสำหรับกดเพื่อค้นหาข้อมูลตามเงื่อนไขที่เลือก       |
| Results Table         | Displays the performance details and calculated results. | ตารางแสดงรายละเอียดผลการดำเนินงานและข้อมูลที่คำนวณได้ |
| Export Button         | Export the evaluation data to an external file.          | ปุ่มสำหรับส่งออกข้อมูลการประเมินผล                    |
| Expand / Collapse All | Toggle all sections in the results table at once.        | ปุ่มสำหรับเปิดหรือปิดการแสดงผลทุกส่วนในตารางพร้อมกัน  |



> #### _To create a Performance Policy_
>
> _navigate to Compliance ▸ Performance ▸_ [_Performance Policy_](../performance/performance-policy.md)_._
