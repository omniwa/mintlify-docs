# Nav Report

### NAV Report

The NAV (Net Asset Value) Report generates asset valuation reports detailing total assets, liabilities and the NAV calculation. It includes a breakdown of equities, bank deposits and accrued fees, NAV per unit and total outstanding units, plus management and custodian charges.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (234).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                             | **คำอธิบาย (ภาษาไทย)**                                  |
| --------------------- | ----------------------------------------------------- | ------------------------------------------------------- |
| Dates                 | Reporting date for the data set.                      | วันที่ที่ต้องการสรุปข้อมูลในรายงาน                      |
| Portfolio Group       | Filter to select a specific group of portfolios.      | ตัวกรองสำหรับเลือกกลุ่มของพอร์ตโฟลิโอ                   |
| Show Closed Portfolio | Include portfolios that have been closed in the list. | ตัวเลือกสำหรับรวมพอร์ตโฟลิโอที่ปิดไปแล้วเข้ามาในรายงาน  |
| Portfolio Available   | List of portfolios available for selection.           | รายชื่อพอร์ตโฟลิโอทั้งหมดที่สามารถเลือกมาออกรายงานได้   |
| Move Button (>)       | Move the highlighted portfolios to the Selected list. | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังรายการ "Selected" |
| Portfolio Selected    | Portfolios chosen to be included in the report.       | รายชื่อพอร์ตโฟลิโอที่จะถูกนำไปประมวลผลในรายงาน          |
| Download Excel Button | Export the generated report to an Excel file.         | ปุ่มสำหรับดาวน์โหลดรายงานในรูปแบบไฟล์ Excel             |
| View PDF Button       | Preview the report in PDF format before downloading.  | ปุ่มสำหรับเรียกดูตัวอย่างรายงานในรูปแบบไฟล์ PDF         |
