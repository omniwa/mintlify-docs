# Daily Performance

### Daily Performance

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (262).png" alt=""><figcaption></figcaption></figure></div>

| **Component**           | **Description (English)**                                        | **คำอธิบาย (ภาษาไทย)**                                           |
| ----------------------- | ---------------------------------------------------------------- | ---------------------------------------------------------------- |
| Download Window         | Window for selecting available download settings.                | หน้าต่างสำหรับตั้งค่าเงื่อนไขการดาวน์โหลดรายงาน                  |
| Date                    | Specific reporting date for performance analysis.                | วันที่ที่ต้องการใช้ในการวิเคราะห์ผลการดำเนินงาน                  |
| Timeframe Mode          | The period or timeframe used for calculation (e.g., Daily, MTD). | รูปแบบช่วงเวลาที่ใช้ในการคำนวณ (เช่น รายวัน หรือตั้งแต่ต้นเดือน) |
| Performance Groups      | Categories used to group portfolios by performance.              | การจัดกลุ่มพอร์ตโฟลิโอตามเกณฑ์วัดผลการดำเนินงาน                  |
| SD (Standard Deviation) | Measure of investment risk or volatility.                        | ค่าเบี่ยงเบนมาตรฐาน (SD) เพื่อวัดความเสี่ยงหรือความผันผวน        |
| Annualized              | Convert the performance figures to a yearly basis.               | การปรับค่าผลตอบแทนหรือความเสี่ยงให้เป็นค่าเฉลี่ยต่อปี            |
| Skip Holidays/Weekends  | Exclude non-trading days from the calculation.                   | ตัวเลือกเพื่อไม่นับรวมวันหยุดและวันเสาร์-อาทิตย์ในการคำนวณ       |
| Export As Excel / PDF   | Choose the desired file format for export.                       | เลือกรูปแบบไฟล์ที่ต้องการส่งออก (Excel หรือ PDF)                 |
| Download PDF Button     | Directly download the performance report as a PDF.               | ปุ่มสำหรับดาวน์โหลดรายงานผลการดำเนินงานเป็นไฟล์ PDF ทันที        |
| Preview Button          | View the report summary on-screen before downloading.            | ปุ่มสำหรับเรียกดูตัวอย่างรายงานบนหน้าจอก่อนทำการดาวน์โหลด        |



### Settings tab

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (263).png" alt=""><figcaption></figcaption></figure></div>

| **Component**               | **Description (English)**                                                         | **คำอธิบาย (ภาษาไทย)**                                                                 |
| --------------------------- | --------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| Performance Groups (Search) | Filter and search for specific performance groups to include in the report.       | ช่องค้นหาและเลือกกลุ่มการวัดผล (Performance Groups) ที่ต้องการให้แสดงในรายงาน          |
| Benchmark Since Inception   | Toggle to include the benchmark performance data from the portfolio's start date. | ตัวเลือกเปิด-ปิด การแสดงข้อมูลเกณฑ์มาตรฐาน (Benchmark) นับตั้งแต่วันจัดตั้งพอร์ตโฟลิโอ |
