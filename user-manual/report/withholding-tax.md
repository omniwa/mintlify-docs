# Withholding Tax

### Withholding Tax

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (81).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                                     | **คำอธิบาย (ภาษาไทย)**                                                       |
| --------------------- | ------------------------------------------------------------- | ---------------------------------------------------------------------------- |
| Report Type           | Select between Withholding Tax Certificate or Summary report. | ประเภทรายงาน: เลือกระหว่างหนังสือรับรองการหักภาษี ณ ที่จ่าย หรือรายงานสรุปผล |
| Dates                 | Reporting date for the tax data.                              | วันที่ที่ต้องการสรุปข้อมูลในรายงานภาษี                                       |
| Portfolio Group       | Filter to select a specific group of portfolios.              | ตัวกรองสำหรับเลือกกลุ่มของพอร์ตโฟลิโอ                                        |
| Show Closed Portfolio | Include portfolios that have been closed in the list.         | ตัวเลือกสำหรับรวมพอร์ตโฟลิโอที่ปิดไปแล้วเข้ามาแสดงผล                         |
| Portfolio Available   | List of portfolios available for selection.                   | รายชื่อพอร์ตโฟลิโอทั้งหมดที่สามารถเลือกได้                                   |
| Move Button (>)       | Move the highlighted portfolios to the Selected list.         | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังรายการ "Selected"                      |
| Portfolio Selected    | Portfolios chosen to be included in the tax report.           | รายชื่อพอร์ตโฟลิโอที่จะถูกนำไปประมวลผลในรายงาน                               |
| Settlement Date       | The date on which the transaction is settled.                 | วันที่ชำระราคาของรายการ                                                      |
| Authorized Person     | The name of the responsible signer for the document.          | ชื่อผู้มีอำนาจลงนามที่รับผิดชอบในเอกสาร                                      |
| Preview Button        | Preview the report before final printing.                     | ปุ่มสำหรับเรียกดูตัวอย่างรายงานก่อนสั่งพิมพ์จริง                             |
| Remark                | Additional notes or comments for the document.                | หมายเหตุหรือคำอธิบายเพิ่มเติมที่จะระบุในเอกสาร                               |
| Print Button          | Generate and print the tax report.                            | ปุ่มสำหรับสั่งพิมพ์รายงานภาษี                                                |
