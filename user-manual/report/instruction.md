# Instruction

The Report module is the place where every regulator-, client- or management-facing document is generated. The chapter covers Instruction reports, NAV reports, the Cash Report, BOT, PF 1000, AIMC, High Watermark, Soft Commission, Performance Fee, Fee Summary, Gain / Losses, Daily Performance and the Performance Fund Policy.



{% tabs %}
{% tab title="Instruction : Print" %}
#### Print

The Instruction Report tracks Buy & Sell, Fees & Charges, Cash In / Out and FX transactions. It guarantees accurate record-keeping and exposes fields for trade date, custodian, instruction number and transaction details.

{% tab title="Instruction : Reprint" %}
#### Reprint

To access previously generated instructions, navigate to the 'Reprint' tab, where a list of already printed documents will be displayed for re-issuance.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (233).png" alt=""><figcaption></figcaption></figure></div>
{% endtab %}

| **Component**         | **Description (English)**                              | **คำอธิบาย (ภาษาไทย)**                                         |
| --------------------- | ------------------------------------------------------ | -------------------------------------------------------------- |
| Trade Date            | Date of the transaction.                               | วันที่ทำรายการซื้อขาย                                          |
| Reprint / Print Type  | Options for reprinting or selecting the document type. | รูปแบบการพิมพ์: ตัวเลือกสำหรับการพิมพ์ซ้ำหรือเลือกประเภทเอกสาร |
| Custodian             | Filter to select a specific Custodian.                 | ตัวกรองสำหรับเลือกผู้ดูแลรับฝากทรัพย์สิน (Custodian)           |
| Instruction Type      | Category of the transaction instruction.               | ประเภทของคำสั่งรายการ (เช่น การซื้อ การขาย)                    |
| Signature             | Authorised signature for the document.                 | รายชื่อผู้มีอำนาจลงนามที่จะให้ปรากฏในเอกสาร                    |
| Remark                | Additional notes or comments.                          | หมายเหตุหรือคำอธิบายเพิ่มเติม                                  |
| Export As Excel / PDF | Choose the export format between Excel or PDF.         | รูปแบบการส่งออก: เลือกระหว่างไฟล์ Excel หรือ PDF               |
| Print PDF Button      | Generate the document in PDF format.                   | ปุ่มสำหรับสร้างและพิมพ์เอกสารในรูปแบบ PDF                      |
| Data Table            | Lists previously viewed or processed transactions.     | ตารางแสดงรายการธุรกรรมที่เคยเรียกดูหรือทำรายการไว้ก่อนหน้า     |
