# Consolidated Report

The Consolidated Report page allows users to generate comprehensive summaries by merging data from multiple portfolios and specific security types. This tool is essential for viewing cross-portfolio exposure and total asset valuation on a single platform.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (57).png" alt=""><figcaption></figcaption></figure></div>

| **Component**          | **Description (English)**                                  | **คำอธิบาย (ภาษาไทย)**                                      |
| ---------------------- | ---------------------------------------------------------- | ----------------------------------------------------------- |
| Date                   | Select the specific reporting date for consolidation.      | เลือกวันที่ที่ต้องการสรุปรายงานรวมข้อมูล                    |
| Portfolio Group        | Filter to select portfolios by their designated group.     | ตัวกรองสำหรับเลือกพอร์ตโฟลิโอตามกลุ่มที่กำหนดไว้            |
| Show Closed Portfolio  | Include portfolios that have been officially closed.       | ตัวเลือกสำหรับรวมพอร์ตโฟลิโอที่ปิดไปแล้วเข้ามาแสดงผล        |
| Portfolios (Available) | List of portfolios available to be added to the report.    | รายชื่อพอร์ตโฟลิโอทั้งหมดที่สามารถเลือกมาทำรายงานได้        |
| Portfolios (Selected)  | Portfolios currently chosen for the consolidated report.   | รายชื่อพอร์ตโฟลิโอที่ถูกเลือกเพื่อนำมาสรุปข้อมูลรวม         |
| Move Button (> / <)    | Transfer items between Available and Selected lists.       | ปุ่มสำหรับย้ายรายการระหว่างช่อง Available และ Selected      |
| Security Type          | Filter securities by category (e.g., Stock-TH, Stock-USD). | ตัวกรองประเภทหลักทรัพย์ (เช่น หุ้นไทย, หุ้นต่างประเทศ)      |
| Securities (Available) | List of individual securities available for selection.     | รายชื่อหลักทรัพย์ทั้งหมดที่มีให้เลือกตามประเภทที่คัดกรอง    |
| Securities (Selected)  | Specific securities chosen to be detailed in the report.   | รายชื่อหลักทรัพย์ที่ถูกเลือกเพื่อนำมาแสดงรายละเอียดในรายงาน |
| Download Excel Button  | Export the consolidated data into an Excel file.           | ปุ่มสำหรับดาวน์โหลดรายงานข้อมูลรวมออกมาเป็นไฟล์ Excel       |
