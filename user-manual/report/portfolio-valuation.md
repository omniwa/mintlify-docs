# Portfolio Valuation

### Portfolio Valuation

Portfolio Valuation provides a detailed breakdown of asset holdings — stock holdings with unit count, cost price, market price, unrealized gain/loss and percentage of NAV; cash equivalents across multiple currencies; total market value by sector; and the NAV calculation showing portfolio performance and unit valuation.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (71).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                             | **คำอธิบาย (ภาษาไทย)**                                  |
| --------------------- | ----------------------------------------------------- | ------------------------------------------------------- |
| Dates                 | Reporting date for the data set.                      | วันที่ที่ต้องการสรุปข้อมูลในรายงาน                      |
| Portfolio Group       | Filter to select a specific group of portfolios.      | ตัวกรองสำหรับเลือกกลุ่มของพอร์ตโฟลิโอ                   |
| Show Closed Portfolio | Include portfolios that have been closed in the list. | ตัวเลือกสำหรับรวมพอร์ตโฟลิโอที่ปิดไปแล้วเข้ามาแสดงผล    |
| Portfolio Available   | List of portfolios available for selection.           | รายชื่อพอร์ตโฟลิโอทั้งหมดที่สามารถเลือกได้              |
| Move Button (>)       | Move the highlighted portfolios to the Selected list. | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังรายการ "Selected" |
| Portfolio Selected    | Portfolios chosen to be included in the report.       | รายชื่อพอร์ตโฟลิโอที่ถูกเลือกเพื่อนำไปประมวลผล          |
| Generate Full Report  | Run and process the full comprehensive report.        | ปุ่มสำหรับเริ่มประมวลผลรายงานฉบับสมบูรณ์                |
| Download Button       | Export the report to Excel or PDF format.             | ปุ่มสำหรับดาวน์โหลดรายงานในรูปแบบไฟล์ Excel หรือ PDF    |
