# BOT Report

### BOT Report

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (237).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                                | **คำอธิบาย (ภาษาไทย)**                                |
| --------------------- | -------------------------------------------------------- | ----------------------------------------------------- |
| Dates                 | Reporting date for the data set.                         | วันที่ที่ต้องการเรียกดูข้อมูลในรายงาน                 |
| Portfolio Group       | Filter to select a specific group of portfolios.         | ตัวกรองสำหรับเลือกกลุ่มของพอร์ตโฟลิโอ                 |
| Show Closed Portfolio | Include portfolios that have been closed in the list.    | ตัวเลือกสำหรับรวมพอร์ตโฟลิโอที่ปิดไปแล้วเข้ามาแสดงผล  |
| Portfolio Available   | List of portfolios available for selection.              | รายชื่อพอร์ตโฟลิโอทั้งหมดที่ระบบอนุญาตให้เลือกได้     |
| Move Button (>)       | Move the highlighted portfolios to the Selected list.    | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังช่อง "Selected" |
| Portfolio Selected    | Portfolios chosen to be included in the export.          | รายชื่อพอร์ตโฟลิโอที่ถูกเลือกเพื่อนำไปออกรายงาน       |
| Download Excel Button | Export the data of selected portfolios to an Excel file. | ปุ่มสำหรับดาวน์โหลดข้อมูลออกเป็นไฟล์ Excel            |
