# Gain Losses

### Gain / Losses

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (258).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                                                     | **คำอธิบาย (ภาษาไทย)**                                                  |
| --------------------- | ----------------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| Realized / Unrealized | Select to show actual gains/losses (Realized) or projected ones (Unrealized). | เลือกแสดงผลกำไร/ขาดทุนที่เกิดขึ้นจริง หรือที่ยังไม่รับรู้ (ตามราคาตลาด) |
| Dates                 | Reporting period for the gain/loss data.                                      | ช่วงเวลาหรือวันที่ที่ต้องการสรุปข้อมูลกำไรและขาดทุน                     |
| Portfolio Group       | Filter to select a specific group of portfolios.                              | ตัวกรองสำหรับเลือกกลุ่มของพอร์ตโฟลิโอ                                   |
| Show Closed Portfolio | Include portfolios that have been closed in the list.                         | ตัวเลือกสำหรับรวมพอร์ตโฟลิโอที่ปิดไปแล้วเข้ามาแสดงผล                    |
| Portfolio Available   | List of portfolios available for selection.                                   | รายชื่อพอร์ตโฟลิโอทั้งหมดที่สามารถเลือกมาออกรายงานได้                   |
| Move Portfolio (>)    | Move the highlighted portfolios to the Selected list.                         | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังรายการ "Selected"                 |
| Portfolio Selected    | Portfolios chosen to be included in the report.                               | รายชื่อพอร์ตโฟลิโอที่จะถูกนำไปประมวลผลในรายงาน                          |
| Export As Excel       | Export the gain/loss data to an Excel file.                                   | ปุ่มสำหรับส่งออกข้อมูลเป็นไฟล์ Excel                                    |
| Export As PDF         | Export the report as a PDF document.                                          | ปุ่มสำหรับส่งออกรายงานในรูปแบบไฟล์ PDF                                  |
| Download Realized PDF | Download the realized gain/loss report directly as a PDF.                     | ปุ่มสำหรับดาวน์โหลดรายงานกำไร/ขาดทุนที่เกิดขึ้นจริงเป็น PDF ทันที       |
