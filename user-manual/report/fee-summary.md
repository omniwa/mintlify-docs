# Fee Summary

### Fee Summary

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (255).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                                                   | **คำอธิบาย (ภาษาไทย)**                                                                                |
| --------------------- | --------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| Fee Type              | Select the category of fees: Management, Custodian, Registrar, or Auditing. | ประเภทค่าธรรมเนียม: เลือกระหว่างค่าธรรมเนียมการจัดการ, ผู้รับฝากทรัพย์สิน, นายทะเบียน หรือการสอบบัญชี |
| Dates                 | Reporting period or specific date for the fee data.                         | ช่วงเวลาหรือวันที่ที่ต้องการสรุปข้อมูลค่าธรรมเนียม                                                    |
| Portfolio Group       | Filter to select a specific group of portfolios.                            | ตัวกรองสำหรับเลือกกลุ่มของพอร์ตโฟลิโอ                                                                 |
| Show Closed Portfolio | Include portfolios that have been closed in the list.                       | ตัวเลือกสำหรับรวมพอร์ตโฟลิโอที่ปิดไปแล้วเข้ามาแสดงผล                                                  |
| Portfolio Available   | List of portfolios available for fee reporting.                             | รายชื่อพอร์ตโฟลิโอทั้งหมดที่สามารถเลือกมาออกรายงานได้                                                 |
| Move Portfolio (>)    | Move the highlighted portfolios to the Selected list.                       | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังรายการ "Selected"                                               |
| Portfolio Selected    | Portfolios chosen to be included in the fee report.                         | รายชื่อพอร์ตโฟลิโอที่จะถูกนำไปประมวลผลในรายงาน                                                        |
| Download Excel Button | Export the selected fee data to an Excel file.                              | ปุ่มสำหรับดาวน์โหลดข้อมูลรายงานค่าธรรมเนียมออกเป็นไฟล์ Excel                                          |
