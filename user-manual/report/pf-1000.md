# PF 1000

### PF 1000

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (239).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                             | **คำอธิบาย (ภาษาไทย)**                                  |
| --------------------- | ----------------------------------------------------- | ------------------------------------------------------- |
| Dates                 | Reporting date for the summary data.                  | วันที่ที่ต้องการสรุปข้อมูลในรายงาน                      |
| Portfolio Group       | Filter to select a specific group of portfolios.      | ตัวกรองสำหรับเลือกกลุ่มของพอร์ตโฟลิโอ                   |
| Show Closed Portfolio | Include portfolios that have been closed in the list. | ตัวเลือกสำหรับรวมพอร์ตโฟลิโอที่ปิดไปแล้วเข้ามาแสดงผล    |
| Portfolio Available   | List of portfolios available for selection.           | รายชื่อพอร์ตโฟลิโอทั้งหมดที่สามารถเลือกได้              |
| Move Button (>)       | Move the highlighted portfolios to the Selected list. | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังรายการ "Selected" |
| Portfolio Selected    | Portfolios chosen to be included in the summary.      | รายชื่อพอร์ตโฟลิโอที่ถูกเลือกเพื่อนำมาประมวลผล          |
| Get Summary Button    | Run and view the summary details on screen.           | ปุ่มสำหรับกดเรียกดูข้อมูลสรุปตามเงื่อนไขที่เลือก        |
| Checkbox (THB)        | Option to convert values to Thai Baht.                | ตัวเลือกสำหรับแปลงมูลค่าให้แสดงผลเป็นสกุลเงินบาท        |



### Other Assets / Payables Table : ตารางรายการสินทรัพย์อื่นและยอดค้างจ่าย

| **Component**  | **Description (English)**                   | **คำอธิบาย (ภาษาไทย)**                       |
| -------------- | ------------------------------------------- | -------------------------------------------- |
| Get Summary    | View summary details of all recorded items. | ปุ่มสำหรับเรียกดูรายละเอียดข้อมูลสรุปทั้งหมด |
| Checkbox (THB) | Convert values to Thai Baht for reporting.  | ตัวเลือกสำหรับแปลงมูลค่าเป็นสกุลเงินบาท      |
| Amount         | The original recorded amount.               | จำนวนเงินตามรายการที่บันทึก                  |
| Adjust Amount  | Modify the recorded amount for corrections. | ส่วนสำหรับปรับปรุงหรือแก้ไขจำนวนเงิน         |
