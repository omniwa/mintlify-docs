# Cash Report

### Cash Report

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (74).png" alt=""><figcaption></figcaption></figure></div>

| **Component**                          | **Description (English)**                             | **คำอธิบาย (ภาษาไทย)**                                  |
| -------------------------------------- | ----------------------------------------------------- | ------------------------------------------------------- |
| Start Dates                            | Reporting date for the data set.                      | วันที่เริ่มต้นสำหรับการสรุปข้อมูลในรายงาน               |
| Portfolio Group                        | Filter to select a specific group of portfolios.      | ตัวกรองสำหรับเลือกกลุ่มของพอร์ตโฟลิโอ                   |
| Currencies                             | Specific currency tracked in the report.              | สกุลเงินที่ใช้ติดตามผลในรายงาน                          |
| Show Closed Portfolio                  | Include portfolios that have been closed in the list. | ตัวเลือกสำหรับรวมพอร์ตโฟลิโอที่ปิดไปแล้วเข้ามาแสดงผล    |
| Portfolio Available                    | List of portfolios available for selection.           | รายชื่อพอร์ตโฟลิโอทั้งหมดที่สามารถเลือกได้              |
| Move Button (>)                        | Move the highlighted portfolios to the Selected list. | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังรายการ "Selected" |
| Portfolio Selected                     | Portfolios chosen to be included in the report.       | รายชื่อพอร์ตโฟลิโอที่ถูกเลือกเพื่อนำไปประมวลผล          |
| Search Button                          | Run the search based on the selected criteria.        | ปุ่มสำหรับกดเพื่อเริ่มค้นหาข้อมูลตามเงื่อนไขที่กำหนด    |
| Cash Management Report                 | Overview of all cash-related transactions.            | รายงานภาพรวมการจัดการธุรกรรมเงินสดทั้งหมด               |
| Cash Account & Accrued Interest Report | Report on account balances and accrued interest.      | รายงานแสดงยอดเงินคงเหลือในบัญชีและดอกเบี้ยค้างรับ       |
| Export Button                          | Download the report data to an external file.         | ปุ่มสำหรับดาวน์โหลดข้อมูลรายงานออกไปยังไฟล์ภายนอก       |
