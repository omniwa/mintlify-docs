# Soft Commission

### Soft Commission

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (246).png" alt=""><figcaption></figcaption></figure></div>

| **Component**            | **Description (English)**                                      | **คำอธิบาย (ภาษาไทย)**                                      |
| ------------------------ | -------------------------------------------------------------- | ----------------------------------------------------------- |
| Create Category Button   | Add a new soft-commission category to the system.              | ปุ่มสำหรับเพิ่มหมวดหมู่ Soft Commission ใหม่                |
| Category                 | Classification or grouping for commissions.                    | ชื่อหมวดหมู่หรือกลุ่มประเภทของค่าธรรมเนียม                  |
| Prefixes                 | Portfolio prefixes (supports special characters such as “\_”). | คำนำหน้าชื่อพอร์ตโฟลิโอ (รองรับอักขระพิเศษ เช่น “\_”)       |
| Item                     | Content shown in the monthly report footer.                    | เนื้อหาหรือรายละเอียดที่จะแสดงในส่วนท้ายของรายงานประจำเดือน |
| Add Button               | Insert a new soft-commission entry.                            | ปุ่มสำหรับเพิ่มรายการ Soft Commission เข้าไปในหมวดหมู่      |
| Edit Button              | Modify the category name and prefix settings.                  | ปุ่มสำหรับแก้ไขชื่อหมวดหมู่และการตั้งค่าคำนำหน้า            |
| Delete Button            | Remove the selected category from the system.                  | ปุ่มสำหรับลบหมวดหมู่ที่ไม่ต้องการออกจากระบบ                 |
| Edit (Text Modification) | Adjust the specific text content within a category.            | ปุ่มสำหรับแก้ไขข้อความรายละเอียดภายในแต่ละหมวดหมู่          |

### Create / Edit Category in Action column

| ---------------- | -------------------------------------------------------- | --------------------------------------------------------- |
| Category Name    | Title or name of the soft-commission category.           | ชื่อหรือหัวข้อของหมวดหมู่ Soft Commission                 |
| Prefix           | Prefix used to identify and group portfolios.            | คำนำหน้าชื่อที่ใช้ในการระบุหรือจัดกลุ่มพอร์ตโฟลิโอ        |
| Add Field Button | Add another prefix field for multiple identifiers.       | ปุ่มสำหรับเพิ่มช่องกรอกคำนำหน้าชื่อพอร์ตโฟลิโอเพิ่มเติม   |
| Cancel Button    | Discard the current entries and close the window.        | ปุ่มสำหรับยกเลิกการกรอกข้อมูลและปิดหน้าต่าง               |
| Save Button      | Save the category and its prefix settings to the system. | ปุ่มสำหรับบันทึกชื่อหมวดหมู่และการตั้งค่าคำนำหน้าลงในระบบ |



### Edit Soft Commission in item column

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (250).png" alt=""><figcaption></figcaption></figure></div>

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (251).png" alt=""><figcaption></figcaption></figure></div>

| **Component** | **Description (English)**                                     | **คำอธิบาย (ภาษาไทย)**                                                   |
| ------------- | ------------------------------------------------------------- | ------------------------------------------------------------------------ |
| COMPANY       | Name of the brokerage or service provider.                    | ชื่อบริษัทหลักทรัพย์หรือผู้ให้บริการ                                     |
| TYPE          | Type of services or benefits received (e.g., News, Research). | ประเภทของบริการหรือสิทธิประโยชน์ที่ได้รับ (เช่น ข้อมูลข่าวสาร, งานวิจัย) |
| REASON        | The objective or justification for the commission.            | เหตุผลหรือวัตถุประสงค์ในการรับบริการ (เพื่อประโยชน์ของกองทุนส่วนบุคคล)   |
| Edit Button   | Modify the details of an existing row.                        | ปุ่มสำหรับแก้ไขรายละเอียดข้อมูลในแถวนั้น ๆ                               |
| Delete Button | Remove the specific row from the category.                    | ปุ่มสำหรับลบรายการนั้นออกจากหมวดหมู่                                     |
| + Add New Row | Insert a new service provider and details.                    | ปุ่มสำหรับเพิ่มรายการผู้ให้บริการใหม่ลงในตาราง                           |
| Cancel Button | Close the edit window without saving changes.                 | ปุ่มสำหรับยกเลิกและปิดหน้าต่างแก้ไขโดยไม่บันทึกการเปลี่ยนแปลง            |
