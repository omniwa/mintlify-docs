# Nav Summary

### NAV Summary

NAV Summary aggregates Net Asset Value data across selected portfolios — including NAV calculations per portfolio, total assets, liabilities, outstanding units, NAV per unit, selling price and redemption price.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (78).png" alt=""><figcaption></figcaption></figure></div>

| **Component**                                                      | **Description (English)**                             | **คำอธิบาย (ภาษาไทย)**                                  |
| ------------------------------------------------------------------ | ----------------------------------------------------- | ------------------------------------------------------- |
| Dates                                                              | Reporting date for the data set.                      | วันที่ที่ต้องการสรุปข้อมูลในรายงาน                      |
| Portfolio Group                                                    | Filter to select a specific group of portfolios.      | ตัวกรองสำหรับเลือกกลุ่มของพอร์ตโฟลิโอ                   |
| Show Closed Portfolio                                              | Include portfolios that have been closed in the list. | ตัวเลือกสำหรับรวมพอร์ตโฟลิโอที่ปิดไปแล้วเข้ามาแสดงผล    |
| Portfolio Available                                                | List of portfolios available for selection.           | รายชื่อพอร์ตโฟลิโอทั้งหมดที่สามารถเลือกมาออกรายงานได้   |
| Move Button (>)                                                    | Move the highlighted portfolios to the Selected list. | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังรายการ "Selected" |
| Portfolio Selected                                                 | Portfolios chosen to be included in the report.       | รายชื่อพอร์ตโฟลิโอที่ถูกเลือกเพื่อนำไปประมวลผล          |
| Search Button                                                      | Run the search based on the selected criteria.        | ปุ่มสำหรับกดเพื่อเริ่มค้นหาข้อมูลตามเงื่อนไขที่กำหนด    |
| Results Table                                                      | Displays the queried data in a structured format.     | ตารางแสดงผลลัพธ์ข้อมูลที่ได้จากการค้นหา                 |
| <p>Download Excel<br>- Layout: Default<br>- Layout: Compliance</p> | Export the search results to an Excel file.           | ปุ่มสำหรับดาวน์โหลดข้อมูลผลลัพธ์ในรูปแบบไฟล์ Excel      |
