# High Watermark

### High Watermark

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (244).png" alt=""><figcaption></figcaption></figure></div>

| **Component**      | **Description (English)**                                           | **คำอธิบาย (ภาษาไทย)**                                          |
| ------------------ | ------------------------------------------------------------------- | --------------------------------------------------------------- |
| Portfolio          | Select a specific portfolio to manage.                              | เลือกพอร์ตโฟลิโอที่ต้องการจัดการข้อมูล                          |
| Data Table         | Displays recorded portfolio information and High Watermark history. | ตารางแสดงข้อมูลรายละเอียดและประวัติ High Watermark ที่บันทึกไว้ |
| Add New Row Button | Insert a new High Watermark record into the system.                 | ปุ่มสำหรับเพิ่มรายการ High Watermark ใหม่เข้าสู่ระบบ            |
| Edit Button        | Modify an existing High Watermark record.                           | ปุ่มสำหรับแก้ไขข้อมูลรายการที่บันทึกไว้แล้ว                     |
| Delete Button      | Remove a record from the High Watermark list.                       | ปุ่มสำหรับลบรายการที่ไม่ต้องการออกจากระบบ                       |
