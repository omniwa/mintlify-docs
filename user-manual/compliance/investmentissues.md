# Investment Issues

### Investment Issues

Use it to approve, reject or revert decisions, and to keep a complete audit trail of approvals.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (133).png" alt=""><figcaption></figcaption></figure></div>

| **Component**                        | **Description (English)**                             | **คำอธิบาย (ภาษาไทย)**                                       |
| ------------------------------------ | ----------------------------------------------------- | ------------------------------------------------------------ |
| Investments                          | Table of investments awaiting approval.               | ตารางแสดงรายการลงทุนที่กำลังรอการอนุมัติ                     |
| Columns                              | Adjust the visible columns to match your role.        | ปุ่มสำหรับเลือกปรับการแสดงผลคอลัมน์ให้เหมาะสมกับบทบาทหน้าที่ |
| Filter                               | Filter the table by status.                           | ตัวกรองสำหรับเลือกดูรายการตามสถานะที่ต้องการ                 |
| ↩ Revert Approval / Rejection Button | Undo a previous Approval or Rejection action.         | ปุ่มสำหรับยกเลิกการอนุมัติหรือการปฏิเสธรายการที่ทำไปก่อนหน้า |
| Approval Log                         | Table of investments that have already been approved. | ตารางประวัติการอนุมัติรายการลงทุนที่ดำเนินการเสร็จสิ้นแล้ว   |
| Date                                 | Date of the action taken.                             | วันที่ที่มีการดำเนินการกับรายการนั้นๆ                        |
| Status                               | Current status of the investment.                     | สถานะปัจจุบันของรายการลงทุน                                  |
