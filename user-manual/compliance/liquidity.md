# Liquidity

### Liquidity

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (175).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                              | **คำอธิบาย (ภาษาไทย)**                                                   |
| --------------------- | ------------------------------------------------------ | ------------------------------------------------------------------------ |
| Dates                 | Reporting date for the data set.                       | วันที่ที่ต้องการสรุปข้อมูลในรายงาน                                       |
| Portfolio Group       | Filter to select a specific group of portfolios.       | ตัวกรองสำหรับเลือกกลุ่มของพอร์ตโฟลิโอ                                    |
| Show Closed Portfolio | Include portfolios that have been closed in the list.  | ตัวเลือกสำหรับรวมพอร์ตโฟลิโอที่ปิดไปแล้วเข้ามาแสดงผล                     |
| Portfolio Available   | List of portfolios that are available for selection.   | รายชื่อพอร์ตโฟลิโอทั้งหมดที่สามารถเลือกมาออกรายงานได้                    |
| Button >              | Move the highlighted portfolio to the Selected list.   | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังรายการ "Selected"                  |
| Portfolio Selected    | Portfolios that have been chosen for the report.       | รายชื่อพอร์ตโฟลิโอที่ถูกเลือกเพื่อนำมาประมวลผล                           |
| Security Type         | Specify the type or category of securities involved.   | ระบุประเภทหรือหมวดหมู่ของหลักทรัพย์ที่ต้องการเรียกดู (เช่น Equity, Bond) |
| Securities Available  | Securities that can be selected based on the criteria. | รายชื่อหลักทรัพย์ที่สามารถเลือกได้ตามเงื่อนไขที่กำหนด                    |
| Securities Selected   | Securities that have been chosen for the report.       | รายชื่อหลักทรัพย์ที่ถูกเลือกเพื่อนำมาแสดงผลในรายงาน                      |
| Download Excel Button | Download the generated report to an Excel file.        | ปุ่มสำหรับดาวน์โหลดรายงานในรูปแบบไฟล์ Excel                              |
