# Monitoring Report

### Monitoring Report

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (104).png" alt=""><figcaption></figcaption></figure></div>

| **Component**                     | **Description (English)**                                 | **คำอธิบาย (ภาษาไทย)**                                                 |
| --------------------------------- | --------------------------------------------------------- | ---------------------------------------------------------------------- |
| Report Type                       | Select the specific type of report to generate.           | เลือกประเภทของรายงานที่ต้องการสร้าง                                    |
| Dates                             | Reporting date for the data.                              | วันที่ที่ต้องการสรุปข้อมูลในรายงาน                                     |
| Start from beginning of the month | Use the first day of the current month as the start date. | เริ่มต้นจากต้นเดือน: กำหนดให้ใช้วันแรกของเดือนเป็นจุดเริ่มต้นของข้อมูล |
| Portfolio Group                   | Filter to select a specific group of portfolios.          | ตัวกรองสำหรับเลือกกลุ่มของพอร์ตโฟลิโอ                                  |
| Show Closed Portfolio             | Include portfolios that have been closed in the list.     | ตัวเลือกสำหรับรวมพอร์ตโฟลิโอที่ปิดไปแล้วเข้ามาในรายงาน                 |
| Portfolios Available              | List of portfolios available for selection.               | รายชื่อพอร์ตโฟลิโอทั้งหมดที่สามารถเลือกมาออกรายงานได้                  |
| Button >                          | Move the highlighted portfolios to the Selected list.     | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังรายการ "Selected"                |
| Portfolios Selected               | Portfolios chosen to be included in the report.           | รายชื่อพอร์ตโฟลิโอที่จะถูกนำไปประมวลผลในรายงาน                         |
| Export As                         | Choose the export format — Excel, PDF, or Text.           | รูปแบบการส่งออก: เลือกไฟล์ที่ต้องการ เช่น Excel, PDF หรือ Text         |
| Download Button                   | Download the generated report in the selected format.     | ปุ่มสำหรับดาวน์โหลดรายงานตามรูปแบบที่เลือกไว้                          |
