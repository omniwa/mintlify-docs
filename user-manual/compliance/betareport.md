# Beta Report

### Beta Report

{% hint style="info" %}
_Related data: Beta values are provided by the Fund Account team and represent each asset’s sensitivity to market movements._
{% endhint %}

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (227).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                             | **คำอธิบาย (ภาษาไทย)**                                  |
| --------------------- | ----------------------------------------------------- | ------------------------------------------------------- |
| Dates                 | Reporting date for the data set.                      | วันที่ที่ต้องการสรุปข้อมูลในรายงาน                      |
| Portfolio Group       | Filter to select a specific group of portfolios.      | ตัวกรองสำหรับเลือกกลุ่มของพอร์ตโฟลิโอ                   |
| Show Closed Portfolio | Include portfolios that have been closed in the list. | ตัวเลือกสำหรับรวมพอร์ตโฟลิโอที่ปิดไปแล้วเข้ามาแสดงผล    |
| Portfolio Available   | List of portfolios that are available for selection.  | รายชื่อพอร์ตโฟลิโอทั้งหมดที่สามารถเลือกมาออกรายงานได้   |
| Button >              | Move the highlighted portfolios to the Selected list. | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังรายการ "Selected" |
| Portfolio Selected    | Portfolios that have been chosen for the report.      | รายชื่อพอร์ตโฟลิโอที่จะถูกนำไปประมวลผลในรายงาน          |
| Search Button         | Run the search based on the selected criteria.        | ปุ่มสำหรับกดเพื่อเริ่มค้นหาข้อมูลตามเงื่อนไขที่กำหนด    |
| Export Excel Button   | Download the generated report to an Excel file.       | ปุ่มสำหรับดาวน์โหลดรายงานในรูปแบบไฟล์ Excel             |
