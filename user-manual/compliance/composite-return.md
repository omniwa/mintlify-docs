# Composite Return

### Composite Return

{% hint style="info" %}
_Related data: Performance NAV — the Net Asset Value performance metrics needed to calculate and compare returns._
{% endhint %}

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (107).png" alt=""><figcaption></figcaption></figure></div>

| **Component**     | **Description (English)**                                 | **คำอธิบาย (ภาษาไทย)**                                           |
| ----------------- | --------------------------------------------------------- | ---------------------------------------------------------------- |
| Date              | Selected date for performance evaluation.                 | วันที่ที่เลือกสำหรับการประเมินผลการดำเนินงาน                     |
| Fund Policies     | Configured under Performance ▸ Performance Policy.        | นโยบายของกองทุน ซึ่งได้รับการตั้งค่าไว้ในเมนู Performance Policy |
| Benchmark Indexes | Reference indexes used for performance comparison.        | ดัชนีเปรียบเทียบ (Benchmark) ที่ใช้เพื่ออ้างอิงผลการดำเนินงาน    |
| Remark            | Additional notes or comments regarding the configuration. | หมายเหตุหรือคำอธิบายเพิ่มเติมเกี่ยวกับการตั้งค่า                 |

### Check Box

Select or deselect specific metrics for the report.

| **Component**              | **Description (English)**                               | **คำอธิบาย (ภาษาไทย)**                                                                  |
| -------------------------- | ------------------------------------------------------- | --------------------------------------------------------------------------------------- |
| SD                         | Standard Deviation — the volatility of returns.         | ส่วนเบี่ยงเบนมาตรฐาน (SD): ค่าที่ใช้วัดความผันผวนของผลตอบแทน                            |
| Annualized                 | Convert returns to a yearly format for comparison.      | การปรับเป็นค่ารายปี: ปรับอัตราผลตอบแทนให้เป็นรูปแบบรายปีเพื่อให้เปรียบเทียบได้ง่ายขึ้น  |
| Skip Holidays and Weekends | Ignore non-trading days in the performance calculation. | ข้ามวันหยุดและวันเสาร์-อาทิตย์: ไม่นับรวมวันหยุดที่ไม่มีการซื้อขายมาคำนวณผลการดำเนินงาน |

| Search Button         | Run the search based on the selected criteria.          | ปุ่มสำหรับกดเพื่อเริ่มทำการค้นหาข้อมูลตามเงื่อนไขที่กำหนดไว้      |
| --------------------- | ------------------------------------------------------- | ----------------------------------------------------------------- |
| Download Excel Button | Download the displayed data or report to an Excel file. | ปุ่มสำหรับดาวน์โหลดข้อมูลหรือรายงานที่ปรากฏอยู่ในรูปแบบไฟล์ Excel |

#### **Composite Return Table**

Table of results returned by the search.

| **Component** | **Description (English)**                                                                 | **คำอธิบาย (ภาษาไทย)**                                                                      |
| ------------- | ----------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| AUM           | Assets Under Management — configured at the Performance Policy level.                     | มูลค่าสินทรัพย์ภายใต้การจัดการ (AUM) ซึ่งตั้งค่าไว้ในระดับนโยบายการวัดผล                    |
| 1M            | Return percentage for the past one-month period.                                          | อัตราผลตอบแทนย้อนหลังในช่วงระยะเวลา 1 เดือนที่ผ่านมา (หน่วยเป็นเปอร์เซ็นต์)                 |
| Policy        | Investment policy applied (e.g., INNO\_XYZ).                                              | นโยบายการลงทุนที่นำมาปรับใช้ (เช่น รหัสนโยบาย INNO\_XYZ)                                    |
| Policy SD     | Volatility of the policy’s returns over time. Higher values indicate greater fluctuation. | ค่าเบี่ยงเบนมาตรฐานของนโยบาย: ใช้วัดความผันผวน หากค่าสูงแสดงว่ามีความเสี่ยงและความผันผวนมาก |
| Benchmark     | Reference index or expected return (e.g., “3% PER ANNUM”).                                | ดัชนีชี้วัดหรือผลตอบแทนคาดหวังที่ใช้ในการเปรียบเทียบ (เช่น 3% ต่อปี)                        |
| Total AUM     | Aggregate value of all assets under management across all policies.                       | มูลค่ารวมของสินทรัพย์ภายใต้การจัดการทั้งหมดจากทุกนโยบาย                                     |
