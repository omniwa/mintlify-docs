# Investment Model

### Investment Model

Models live under Resources ▸ Portfolio ▸ Portfolio Model and can be applied directly inside an investment.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (268).png" alt=""><figcaption></figcaption></figure></div>

| **Component**           | **Description (English)**                                    | **คำอธิบาย (ภาษาไทย)**                                        |
| ----------------------- | ------------------------------------------------------------ | ------------------------------------------------------------- |
| Investment Models Table | Lists all available investment models in the system.         | ตารางแสดงรายชื่อโมเดลการลงทุนทั้งหมดที่มีอยู่ในระบบ           |
| Edit Button             | Modify the parameters or details of an existing model.       | ปุ่มสำหรับแก้ไขการตั้งค่าหรือรายละเอียดของโมเดลเดิมที่มีอยู่  |
| Preview Button          | View detailed information and asset allocation of the model. | ปุ่มสำหรับเรียกดูรายละเอียดเชิงลึกและสัดส่วนสินทรัพย์ของโมเดล |
| Create Investment Model | Open the form to define and add a new investment model.      | ปุ่มสำหรับสร้างและกำหนดรายละเอียดโมเดลการลงทุนใหม่            |

{% hint style="info" %}
_Tip: Models are created at Resources ▸ Portfolio ▸ Portfolio Model._
{% endhint %}

### Edit Investment Model

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (269).png" alt=""><figcaption></figcaption></figure></div>

| **Component**    | **Description (English)**                                   | **คำอธิบาย (ภาษาไทย)**                                                     |
| ---------------- | ----------------------------------------------------------- | -------------------------------------------------------------------------- |
| Model Name       | The designated name of the investment model.                | ชื่อของโมเดลการลงทุน                                                       |
| Members          | List of specific portfolios associated with this model.     | รายชื่อพอร์ตโฟลิโอที่เลือกใช้โมเดลการลงทุนนี้                              |
| Add Group        | Create a new asset group category (e.g., Thailand, Global). | ปุ่มสำหรับเพิ่มกลุ่มสินทรัพย์ใหม่ (เช่น กลุ่มหุ้นไทย, กลุ่มหุ้นต่างประเทศ) |
| Normalize        | Automatically adjust percentages to total 100%.             | ปุ่มสำหรับปรับสัดส่วนตัวเลขให้รวมกันได้ 100% โดยอัตโนมัติ                  |
| Reset            | Revert all changes to the original state.                   | ปุ่มสำหรับล้างค่าที่แก้ไขและกลับไปใช้ค่าเริ่มต้น                           |
| Group Percentage | Enable or disable the display of group-level weighting.     | ตัวเลือกสำหรับเปิดหรือปิดการแสดงผลสัดส่วนร้อยละในระดับกลุ่ม                |
| Previous (%)     | The existing percentage allocation before any updates.      | สัดส่วนเปอร์เซ็นต์เดิมที่ตั้งค่าไว้ก่อนการแก้ไข                            |
| Update To (%)    | Input field to define the new target allocation percentage. | ช่องสำหรับกรอกสัดส่วนเปอร์เซ็นต์ใหม่ที่ต้องการ                             |
| Remaining (%)    | Shows the balance remaining to reach a 100% total.          | แสดงสัดส่วนเปอร์เซ็นต์ที่เหลืออยู่เพื่อให้ครบ 100%                         |
| Action Icons     | Add (+) or Remove (Trash) items and groups.                 | ไอคอนสำหรับเพิ่มรายการ (+), ย้ายลำดับ หรือลบรายการ (ถังขยะ) ออกจากกลุ่ม    |
| Save Changes     | Commit and save the new model configuration.                | ปุ่มสำหรับบันทึกการเปลี่ยนแปลงทั้งหมดเข้าสู่ระบบ                           |
