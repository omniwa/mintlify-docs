# Equity Holding

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (151).png" alt=""><figcaption></figcaption></figure></div>

| **Component**            | **Description (English)**                                                     | **คำอธิบาย (ภาษาไทย)**                                                                                   |
| ------------------------ | ----------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| Date                     | Select the specific date for the holding report.                              | เลือกวันที่ที่ต้องการเรียกดูรายงานยอดคงเหลือหลักทรัพย์                                                   |
| Export Button            | Download the displayed holding data.                                          | ปุ่มสำหรับดาวน์โหลดข้อมูลรายการหลักทรัพย์ในรูปแบบไฟล์                                                    |
| Select a Portfolio Group | Choose a predefined group of portfolios.                                      | เลือกกลุ่มของพอร์ตโฟลิโอที่ต้องการเรียกดูข้อมูล                                                          |
| Show Closed Portfolio    | Include portfolios that have been closed in the report.                       | เลือกให้แสดงพอร์ตโฟลิโอที่ปิดไปแล้วรวมในรายงานด้วย                                                       |
| Portfolio Available      | List of portfolios available to be selected.                                  | รายชื่อพอร์ตโฟลิโอที่สามารถเลือกนำมาแสดงผลได้                                                            |
| > Button                 | Move the highlighted portfolios to the "Selected" list.                       | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังช่อง "Selected"                                                    |
| Portfolio Selected       | Portfolios that will be included in the report.                               | รายชื่อพอร์ตโฟลิโอที่จะถูกนำไปประมวลผลในรายงาน                                                           |
| Columns                  | Adjust and customize which columns are visible.                               | ปรับแต่งและเลือกคอลัมน์ที่ต้องการให้แสดงในตาราง                                                          |
| Grouping                 | Group the holding rows into specific categories.                              | จัดกลุ่มรายการในตารางตามหมวดหมู่ที่ต้องการ                                                               |
| Equity Holding Table     | Lists holdings including security, sector, currency, units, and market value. | ตารางแสดงรายการหลักทรัพย์ ซึ่งรวมถึง รหัสหลักทรัพย์, กลุ่มอุตสาหกรรม, สกุลเงิน, จำนวนหน่วย และมูลค่าตลาด |





{% hint style="info" %}
EOD Requirement: Data will only be displayed once the portfolio has completed the End-of-Day (EOD) process.
{% endhint %}
