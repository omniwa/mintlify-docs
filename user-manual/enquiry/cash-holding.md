# Cash Holding

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (121).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                                    | **คำอธิบาย (ภาษาไทย)**                                               |
| --------------------- | ------------------------------------------------------------ | -------------------------------------------------------------------- |
| Date                  | Select the specific date for the cash holding report.        | เลือกวันที่ที่ต้องการเรียกดูรายงานยอดเงินคงเหลือ                     |
| Portfolio Group       | Filter or categorize by portfolio group.                     | กรองข้อมูลตามกลุ่มของพอร์ตโฟลิโอ                                     |
| Currencies            | Select from available currencies to display.                 | เลือกสกุลเงินที่ต้องการให้แสดงผลในรายงาน                             |
| Show Closed Portfolio | Include portfolios that have been closed in the report.      | เลือกให้แสดงพอร์ตโฟลิโอที่ปิดไปแล้วรวมในรายงานด้วย                   |
| Portfolio Available   | List of portfolios available for selection.                  | รายชื่อพอร์ตโฟลิโอที่สามารถเลือกนำมาเรียกดูข้อมูลได้                 |
| >< Button             | Move portfolios between "Available" and "Selected" lists.    | ปุ่มสำหรับย้ายพอร์ตโฟลิโอไปมาสถานะระหว่าง "Available" และ "Selected" |
| Portfolio Selected    | Portfolios that are currently selected for the report.       | รายชื่อพอร์ตโฟลิโอที่จะถูกนำมาแสดงผลในรายงานปัจจุบัน                 |
| Search Button         | Retrieve and display the cash-holding data based on filters. | ปุ่มสำหรับกดเรียกดูข้อมูลยอดเงินคงเหลือตามเงื่อนไขที่ตั้งไว้         |
| Export Button         | Download the cash-holding data in a file format.             | ปุ่มสำหรับดาวน์โหลดข้อมูลยอดเงินคงเหลือออกมาเป็นไฟล์                 |
| Cash Holding Table    | Displays detailed cash balance information per portfolio.    | ตารางแสดงรายละเอียดของยอดเงินคงเหลือแยกตามแต่ละพอร์ตโฟลิโอ           |





{% hint style="info" %}
EOD Requirement: Data will only be displayed once the portfolio has completed the End-of-Day (EOD) process.
{% endhint %}
