# History Transaction

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (152).png" alt=""><figcaption></figcaption></figure></div>

| **Component**            | **Description (English)**                                    | **คำอธิบาย (ภาษาไทย)**                                                                 |
| ------------------------ | ------------------------------------------------------------ | -------------------------------------------------------------------------------------- |
| Trade / Settle Date Type | Choose between filtering by trade date or settle date.       | เลือกประเภทวันที่ระหว่าง วันที่ทำรายการ (Trade Date) หรือ วันที่ชำระราคา (Settle Date) |
| Date                     | Select the specific date or range for the report.            | เลือกวันที่ที่ต้องการเรียกดูข้อมูล                                                     |
| Export Button            | Export the displayed data (e.g., to Excel or CSV).           | ปุ่มสำหรับส่งออกข้อมูล (เช่น ไฟล์ Excel หรือ CSV)                                      |
| Issuer (Filter)          | Filter transactions by the entity that issued the security.  | กรองรายการตามผู้ออกหลักทรัพย์                                                          |
| Broker (Filter)          | Filter transactions by the executing broker.                 | กรองรายการตามโบรกเกอร์ที่ดำเนินการ                                                     |
| Portfolios (Filter)      | Filter to show data for specific portfolios.                 | กรองข้อมูลตามพอร์ตโฟลิโอที่เลือก                                                       |
| Securities (Filter)      | Filter by specific security codes or names.                  | กรองตามรหัสหรือชื่อหลักทรัพย์                                                          |
| Tx Type (Filter)         | Filter by transaction type (e.g., Buy, Sell).                | กรองตามประเภทธุรกรรม (เช่น ซื้อ, ขาย)                                                  |
| Counterparty (Filter)    | Filter by the trading counterparty.                          | กรองตามคู่สัญญาที่ทำรายการด้วย                                                         |
| Security Class (Filter)  | Filter by asset class (e.g., Equity, Debt).                  | กรองตามชั้นของสินทรัพย์ (เช่น ทุน, หนี้)                                               |
| Security Type (Filter)   | Filter by specific category of security.                     | กรองตามประเภทของหลักทรัพย์                                                             |
| Status (Filter)          | Filter transactions by their current status.                 | กรองรายการตามสถานะ (เช่น Pending, Confirmed)                                           |
| Columns                  | Select and customize which columns are visible in the table. | เลือกและปรับแต่งคอลัมน์ที่ต้องการให้แสดงในตาราง                                        |
| Grouping                 | Group the rows into categories (e.g., group by Portfolio).   | จัดกลุ่มรายการตามหมวดหมู่ (เช่น จัดกลุ่มตามพอร์ตโฟลิโอ)                                |
| Transaction Table        | Main display area listing transaction details.               | ตารางหลักที่แสดงรายละเอียดรายการธุรกรรมทั้งหมด                                         |
| Preview Button           | Open a preview window for a specific transaction row.        | ปุ่มสำหรับดูตัวอย่างรายละเอียดของรายการนั้น ๆ ในหน้าต่างย่อย                           |
