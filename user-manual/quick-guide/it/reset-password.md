# Reset Password —

Navigate to System Management ▸ [User Management](../../system-management/user-management.md).&#x20;

{% stepper %}
{% step %}
Click <a class="button secondary" data-icon="pen-line">Edit</a>  next to the user that needs the reset.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (204).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
Click <a class="button secondary">Reset Password</a> to start the reset process. The system automatically sends a “set new password” link to the user’s registered email address.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (205).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
Click <a class="button secondary">Confirm</a> on the confirmation pop-up to confirm the action.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (206).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
The user clicks <a class="button secondary">Reset Password</a>  in the email to open the password-setup page and chooses a new password.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (207).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
The link will redirect the user to the password setup page.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (208).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
After setting the new password, click <a class="button primary">Submit</a> and Done.
{% endstep %}
{% endstepper %}
