# FX Trade —

### FX Trade — Walkthrough

Use this end-to-end walkthrough to buy a foreign currency for a portfolio.

### Role Fund Manager

{% stepper %}
{% step %}
#### &#x20;Navigate to FX ▸ FX Trade

click <a class="button secondary">+ Create Transaction</a> on the top right, to start a new transaction.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (112).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### **Fill in the trade details**

Complete every required field,

review the values in the Summary box,

then click <a class="button secondary">Create</a> to submit the transaction for confirmation.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (113).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### After Re-check the Details, Click <a class="button secondary">create</a>

Users must carefully review the Summary section to ensure the transaction aligns with the intended requirements. Verify the flow of funds from the source account (From) to the destination account (To).

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (114).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### Click <a class="button secondary" data-icon="check"></a> to confirm the transaction

The transaction will be forwarded to the **operations** queue.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (115).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}
{% endstepper %}



### Role: Operation / Fund Account

{% stepper %}
{% step %}
#### Navigate to FX ▸ Post Transaction&#x20;

Click <a class="button secondary" data-icon="check"></a> to post the confirmed FX trade.&#x20;

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (136).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### After posting, the transaction appears in FX ▸ Close Transaction.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (137).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### Navigate to Daily ▸ Process EOD,&#x20;

Select the relevant Portfolio and click <a class="button secondary">Process EOD</a> to execute the EOD.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (138).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}
{% endstepper %}
