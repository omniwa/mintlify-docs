# Investment Model —



From the left Navbar, go to Investment ▸ Investment Model.

{% stepper %}
{% step %}
#### From the left Navbar, go to Investment ▸ Investment Model.

Click <a class="button secondary" data-icon="pen-line"></a> to edit the model.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (270).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### Click <a class="button secondary">Add Group</a> to add new investment group

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (272).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### Click <a class="button secondary" data-icon="plus"></a> to add a stock/Equity to the model.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (273).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### Choose the equities you want to add from the list.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (274).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### Set the target percentages in the "Update To" field and click <a class="button secondary">Save Change</a>

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (276).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
After the changes are saved, the updated allocations will appear on the Edit Investment page.
{% endstep %}
{% endstepper %}
