# Investment Issues —

Navigate to Compliance ▸ [Investment Issues](../../compliance/investmentissues.md)

to see all investments awaiting approval.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (209).png" alt=""><figcaption></figcaption></figure></div>



### Step to approval or rejection

{% stepper %}
{% step %}
#### Click <a class="button secondary" data-icon="pen-line"></a> to review and verify the details of an investment.

The investment details appear. Carefully review the conditions.&#x20;
{% endstep %}

{% step %}
#### Click <a class="button secondary">Approve</a> or <a class="button secondary">Reject</a>and enter your reasoning in the Remark field.

<div><figure><img src="../../.gitbook/assets/image (210).png" alt=""><figcaption></figcaption></figure> <figure><img src="../../.gitbook/assets/image (211).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### Click <a class="button secondary">Approve Investment</a>or <a class="button secondary">Reject Investment</a> to confirm.

After the action, the result is displayed on the Investment Issues page.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (212).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}
{% endstepper %}



### Reverse the approval or rejection

{% stepper %}
{% step %}
#### Click <a class="button secondary" data-icon="arrow-u-turn-up-left"></a> on the selected investment.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (213).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### Press <a class="button secondary">Revert Approval</a> or <a class="button secondary">Revert Rejection</a> to modify or revoke your decision.

<div><figure><img src="../../.gitbook/assets/image (214).png" alt=""><figcaption></figcaption></figure> <figure><img src="../../.gitbook/assets/image (215).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}
{% endstepper %}

### In case, Revert Restrictions

If the system displays the message:&#x20;

{% hint style="warning" %}
_"The Dealer had already started working on some orders generated from this investment. To revert your verdict, the Dealer must first fully rewind their works of all orders generated from this investment to the initial state."_&#x20;
{% endhint %}

it indicates that the investment decision cannot be immediately reverted.
