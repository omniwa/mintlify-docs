# Composite Return —

#### Composite Return — Walkthrough

Step 1 — Set up a Fund Policy. Navigate to Compliance ▸ Performance ▸ Performance Policy and click (1) to create a new Performance Policy. Click (2) on the new policy and (3) to assign portfolios to it.

<figure><img src="../../.gitbook/assets/ชื่อเรื่อง Page 59 - คำอธิบาย Figure 2.9c — Edit Performance Policy" alt=""><figcaption></figcaption></figure>

Figure 2.9c — Edit Performance Policy

Step 2 — Create a Benchmark Index. Go to Compliance ▸ Performance ▸ Benchmark Index, then create the index.

<figure><img src="../../.gitbook/assets/ชื่อเรื่อง Page 60 - คำอธิบาย Figure 2.9d — Benchmark Index (Create)" alt=""><figcaption></figcaption></figure>

Figure 2.9d — Benchmark Index (Create)

Step 3 — Map portfolios to the benchmark. Go to Compliance ▸ Performance ▸ Portfolio Benchmark Index, click (3) on the chosen benchmark and (4) to attach portfolios.

Step 4 — Enter Benchmark Index Returns daily. Go to Compliance ▸ Performance ▸ Benchmark Index Return, click (1) to create a new entry, fill in all required values and click (2) to save.

<figure><img src="../../.gitbook/assets/ชื่อเรื่อง Page 61 - คำอธิบาย Figure 2.9f — Benchmark Index Return interface" alt=""><figcaption></figcaption></figure>

Figure 2.9f — Benchmark Index Return interface

_Benchmark Index Return values must be entered every business day, starting from the portfolio’s inception. Missing values lead to inaccurate performance comparisons._
