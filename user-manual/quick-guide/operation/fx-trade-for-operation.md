# FX Trade for Operation —

### FX Trade — Walkthrough

Use this end-to-end walkthrough to buy a foreign currency for a portfolio.



### Role: Operation / Fund Account

{% stepper %}
{% step %}
#### Navigate to FX ▸ Post Transaction&#x20;

Click <a class="button secondary" data-icon="check"></a> to post the confirmed FX trade.&#x20;

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (136).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### After posting, the transaction appears in FX ▸ Close Transaction.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (137).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### Navigate to Daily ▸ Process EOD

Select the relevant Portfolio and click <a class="button secondary">Process EOD</a> to execute the EOD.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (138).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}
{% endstepper %}

{% hint style="warning" %}
Data Accuracy

Please ensure that the End-of-Day (EOD) process is performed on the same date the transaction is initiated.
{% endhint %}

{% hint style="info" %}
System Update

The system will only update and reflect the latest transaction data in the portfolio reports once the EOD for that specific date is completed.
{% endhint %}
