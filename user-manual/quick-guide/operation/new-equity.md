# New Equity —

{% stepper %}
{% step %}
### Create the Company

Navigate to Resources ▸ Company Group ▸ Company.&#x20;

Click <a class="button secondary">Create Company</a>&#x20;

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (83).png" alt=""><figcaption></figcaption></figure></div>

Fill in the details, click <a class="button secondary">Create</a>

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (84).png" alt=""><figcaption></figcaption></figure></div>

Review & Prove the action in Pending Company table.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (85).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
### Create the Issuer

Navigate to Resources ▸ Company Group ▸ Issuer.

Click <a class="button secondary">Create Issuer</a>

<figure><img src="../../.gitbook/assets/image (87).png" alt=""><figcaption></figcaption></figure>

Fill in the details, click <a class="button secondary">Create</a>

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (86).png" alt=""><figcaption></figcaption></figure></div>

Review & Prove the action in Pending Issuer table.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (88).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
### Create the Equity

Navigate to Resources ▸ Company Group ▸ Equity.

Click <a class="button secondary">Create Equity</a>

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (91).png" alt=""><figcaption></figcaption></figure></div>

Fill in the details and click <a class="button secondary">Create</a>

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (89).png" alt=""><figcaption></figcaption></figure></div>

Review & Prove the action in Pending Equity table.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (92).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
### Create the Price Code

A pop-up warning appears, prompting you to create a Price Code that matches the new Security Code.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (93).png" alt=""><figcaption></figcaption></figure></div>

Navigate to Resources ▸ Market ▸ Price Code.

The system has automatically created a Pending Price Code&#x20;

Click <a class="button secondary" data-icon="pen-line"></a> to review.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (94).png" alt=""><figcaption></figcaption></figure></div>

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (95).png" alt=""><figcaption></figcaption></figure></div>

After click <a class="button secondary">Update</a>, Back to Pending Price Code and Prove it.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (96).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
### Import the equity price

Navigate to Daily ▸ Equity Price Import.

Import the new Equity price via Import mode or Input mode.

<div><figure><img src="../../.gitbook/assets/image (97).png" alt=""><figcaption></figcaption></figure> <figure><img src="../../.gitbook/assets/image (98).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}
{% endstepper %}
