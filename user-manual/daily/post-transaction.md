# Post Transaction

### Post Transaction

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (144).png" alt=""><figcaption></figcaption></figure></div>

| **Component**          | **Description (English)**                                            | **คำอธิบาย (ภาษาไทย)**                                              |
| ---------------------- | -------------------------------------------------------------------- | ------------------------------------------------------------------- |
| Transaction Trade Date | Filter or search by the date the transaction was executed.           | ช่องสำหรับเลือกหรือค้นหาตามวันที่ทำรายการ                           |
| Columns                | Adjust and customize the visible data columns in the table.          | เมนูสำหรับเลือกแสดงผลคอลัมน์ข้อมูลที่ต้องการ                        |
| Filters                | Refine the displayed transaction data based on specific criteria.    | เครื่องมือกรองข้อมูลเพื่อคัดเลือกเฉพาะรายการที่ต้องการตรวจสอบ       |
| Grouping               | Group transactions (e.g., by Portfolio or Asset Class) for analysis. | ฟังก์ชันจัดกลุ่มรายการธุรกรรมเพื่อการวิเคราะห์และตรวจสอบที่ง่ายขึ้น |
| Table Display          | Lists all completed orders and currently executed transactions.      | ตารางแสดงรายการคำสั่งซื้อขายที่สำเร็จและธุรกรรมที่เกิดขึ้น          |
| Post Button            | Finalize and post the transaction to the Closed Transaction menu.    | ปุ่มสำหรับยืนยันและส่งรายการเข้าสู่เมนูธุรกรรมที่ปิดรายการแล้ว      |
| Edit Button            | Modify transaction details if corrections are needed before posting. | ปุ่มสำหรับแก้ไขรายละเอียดธุรกรรมก่อนทำการยืนยัน (Post)              |
| Preview Button         | View full transaction details and linked accounting entries.         | ปุ่มสำหรับเรียกดูรายละเอียดทั้งหมดและรายการบัญชีที่เกี่ยวข้อง       |



{% hint style="warning" %}
Don't forget to review the Pending Transactions periodically to ensure no trade is left unposted, which could cause discrepancies in your daily NAV reports.
{% endhint %}
