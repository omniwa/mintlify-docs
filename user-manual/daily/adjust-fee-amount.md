# Adjust Fee Amount

### Adjust Fee Amount

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (329).png" alt=""><figcaption></figcaption></figure></div>

| **Component**           | **Description (English)**                                           | **คำอธิบาย (ภาษาไทย)**                              |
| ----------------------- | ------------------------------------------------------------------- | --------------------------------------------------- |
| Portfolio (Filter)      | Select the specific portfolio to review its fees.                   | เลือกพอร์ตโฟลิโอที่ต้องการตรวจสอบค่าธรรมเนียม       |
| Fee Code (Filter)       | Filter by specific fee types (e.g., Management Fee, Custodian Fee). | เลือกกรองตามรหัสประเภทค่าธรรมเนียม                  |
| Fee Date (Filter)       | Specify the date range for fee records.                             | เลือกช่วงวันที่ที่ต้องการตรวจสอบข้อมูล              |
| Daily Fee (Exc VAT)     | The system-calculated fee amount excluding taxes.                   | จำนวนค่าธรรมเนียมรายวันที่ระบบคำนวณให้ (ไม่รวม VAT) |
| Adj Daily Fee (Exc VAT) | Field to manually adjust the daily fee amount.                      | ช่องสำหรับปรับปรุงยอดค่าธรรมเนียมรายวันด้วยตนเอง    |
| VAT / Adj VAT           | Current VAT amount and its adjustment field.                        | ยอดภาษีมูลค่าเพิ่มและช่องสำหรับปรับปรุงยอดภาษี      |
| Total Fee               | The final sum of the adjusted fee and VAT.                          | ยอดรวมค่าธรรมเนียมและภาษีทั้งหมดหลังการปรับปรุง     |
| Operation (Edit)        | Button to enable editing for a specific fee record.                 | ปุ่มสำหรับเริ่มแก้ไขข้อมูลค่าธรรมเนียมในแถวนั้นๆ    |
