# Close Transaction

### Close Transaction

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (21).png" alt=""><figcaption></figcaption></figure></div>

| **Component**          | **Description (English)**                                             | **คำอธิบาย (ภาษาไทย)**                                            |
| ---------------------- | --------------------------------------------------------------------- | ----------------------------------------------------------------- |
| Transaction Trade Date | Filter or search for posted transactions by execution date.           | ช่องสำหรับเลือกหรือค้นหาตามวันที่ทำรายการที่ปิดรายการแล้ว         |
| Columns                | Customize and manage the columns displayed in the table.              | เมนูสำหรับเลือกแสดงผลคอลัมน์ข้อมูลที่ต้องการดู                    |
| Filters                | Sort and narrow down transactions based on specific criteria.         | เครื่องมือกรองข้อมูลเพื่อค้นหารายการที่ต้องการตรวจสอบย้อนหลัง     |
| Grouping               | Aggregate related transactions for easier overview and analysis.      | ฟังก์ชันจัดกลุ่มรายการ (เช่น ตามสกุลเงิน หรือประเภทหลักทรัพย์)    |
| Table Display          | Comprehensive list of all successfully posted transactions.           | ตารางแสดงรายการธุรกรรมทั้งหมดที่ได้รับการยืนยัน (Post) แล้ว       |
| Unpost Button          | Revert a transaction back to the Post Transaction menu for review.    | ปุ่มสำหรับยกเลิกการ Post เพื่อส่งรายการกลับไปแก้ไขหรือตรวจสอบใหม่ |
| Preview Button         | View detailed transaction information and its final accounting entry. | ปุ่มสำหรับเรียกดูรายละเอียดธุรกรรมและบันทึกทางบัญชีที่สิ้นสุดแล้ว |
