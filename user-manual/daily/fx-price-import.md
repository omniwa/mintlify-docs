# Fx Price Import

### FX Price Import

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (326).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                                   | **คำอธิบาย (ภาษาไทย)**                                                |
| --------------------- | ----------------------------------------------------------- | --------------------------------------------------------------------- |
| Upload Area           | Click or drag an `.XLSX` file to start the FX price import. | พื้นที่สำหรับคลิกหรือลากไฟล์ Excel (.xlsx) เพื่อนำเข้าอัตราแลกเปลี่ยน |
| Price Source (Filter) | Select the source of the FX data (e.g., Bloomberg).         | เลือกแหล่งที่มาของข้อมูลอัตราแลกเปลี่ยน                               |
| Date (Filter)         | The specific date for which the FX rates apply.             | วันที่ที่ต้องการบันทึกข้อมูลอัตราแลกเปลี่ยน                           |
| Upload Button         | Confirm the file upload and transfer data to the system.    | ปุ่มสำหรับยืนยันการนำข้อมูลจากไฟล์เข้าสู่ระบบ                         |
| Download Example File | Download the required template for FX price data.           | ลิงก์สำหรับดาวน์โหลดไฟล์ตัวอย่างเพื่อให้กรอกข้อมูลได้ถูกโครงสร้าง     |
| Commodity (Column)    | The base currency (e.g., USD, CNY, EUR).                    | สกุลเงินหลัก (สกุลเงินต้นทาง)                                         |
| Quoted (Column)       | The target currency (e.g., THB, CNY).                       | สกุลเงินที่ใช้เทียบ (สกุลเงินปลายทาง)                                 |
| Mid (Column)          | The median exchange rate used for valuation.                | อัตราแลกเปลี่ยนกลาง (ใช้อ้างอิงสำหรับการประเมินมูลค่า)                |
| Bid / Offer           | The buying and selling rates from the source.               | ราคาเสนอซื้อและราคาเสนอขายจากแหล่งข้อมูล                              |
