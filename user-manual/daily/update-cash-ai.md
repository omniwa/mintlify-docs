# Update Cash AI

### Update Cash AI

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (319).png" alt=""><figcaption></figcaption></figure></div>

| **Component**        | **Description (English)**                                | **คำอธิบาย (ภาษาไทย)**                                 |
| -------------------- | -------------------------------------------------------- | ------------------------------------------------------ |
| Portfolio            | Dropdown to select the target portfolio for AI review.   | เลือกพอร์ตโฟลิโอที่ต้องการตรวจสอบดอกเบี้ยค้างรับ       |
| Security             | Select the specific cash account or savings security.    | เลือกบัญชีเงินสดหรือหลักทรัพย์เงินฝากที่ต้องการจัดการ  |
| Cash AI Date         | Select the date range to display interest records.       | เลือกช่วงเวลาที่ต้องการตรวจสอบข้อมูลดอกเบี้ย           |
| Search Button        | Fetch records based on the selected portfolio and dates. | ปุ่มสำหรับค้นหาข้อมูลตามเงื่อนไขที่ระบุ                |
| Value Date           | The effective date for each interest calculation row.    | วันที่มีผลบังคับใช้ (วันที่ของยอดดอกเบี้ย)             |
| Cash Balance         | The outstanding balance used for interest calculation.   | ยอดเงินคงเหลือในบัญชีที่ใช้ในการคำนวณดอกเบี้ย          |
| Interest Rate        | The daily interest rate applied to the balance.          | อัตราดอกเบี้ยรายวันที่ระบบนำมาคำนวณ                    |
| Daily AI             | The system-calculated accrued interest for that day.     | ยอดดอกเบี้ยค้างรับที่ระบบคำนวณให้โดยอัตโนมัติ          |
| Adj Daily AI         | Field to manually adjust the accrued interest amount.    | ช่องสำหรับกรอกเพื่อปรับปรุงยอดดอกเบี้ยค้างรับด้วยตนเอง |
| Edit / Save / Cancel | Control buttons to modify and confirm AI adjustments.    | ปุ่มสำหรับแก้ไข บันทึก หรือยกเลิกการปรับปรุงข้อมูล     |
