# Fx Price Adjustment

### FX Price Adjustment

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (327).png" alt=""><figcaption></figcaption></figure></div>

| **Component**               | **Description (English)**                              | **คำอธิบาย (ภาษาไทย)**                           |
| --------------------------- | ------------------------------------------------------ | ------------------------------------------------ |
| Price Date (Filter)         | Select the date range to view historical FX rates.     | เลือกช่วงวันที่ที่ต้องการดูข้อมูลอัตราแลกเปลี่ยน |
| Price Source (Filter)       | Filter by data provider (e.g., Bloomberg).             | เลือกกรองตามแหล่งที่มาของข้อมูล                  |
| Commodity / Quoted (Filter) | Search for specific currency pairs (e.g., USD to THB). | เลือกกรองตามคู่สกุลเงินที่ต้องการ                |
| Date (Column)               | The specific trading date for the FX rate.             | วันที่ของอัตราแลกเปลี่ยน                         |
| Price Source (Column)       | The origin of the displayed exchange rate.             | แหล่งที่มาของข้อมูลในแถวนั้น ๆ                   |
| Commodity (Base)            | The base currency being converted from.                | สกุลเงินหลัก (สกุลเงินต้นทาง)                    |
| Quoted (Target)             | The target currency being converted to.                | สกุลเงินที่ใช้เทียบ (สกุลเงินปลายทาง)            |
| Mid Price                   | The current middle rate used for system calculations.  | อัตราแลกเปลี่ยนกลางที่ระบบใช้คำนวณ ณ ปัจจุบัน    |
| Edit Button                 | Enable manual adjustment for a specific currency pair. | ปุ่มสำหรับแก้ไขอัตราแลกเปลี่ยนในแถวที่เลือก      |
