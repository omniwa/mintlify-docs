# Adjust Available Unit

### Adjust Available Unit

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (331).png" alt=""><figcaption></figcaption></figure></div>

| **Component**           | **Description (English)**                                       | **คำอธิบาย (ภาษาไทย)**                             |
| ----------------------- | --------------------------------------------------------------- | -------------------------------------------------- |
| Value Date              | The specific date for which the units are being adjusted.       | วันที่ที่มีผลในการปรับปรุงจำนวนหน่วยลงทุน          |
| Portfolio Code          | Unique identifier for each individual portfolio.                | รหัสพอร์ตโฟลิโอที่ต้องการจัดการหน่วยลงทุน          |
| Today's NAV             | Current Net Asset Value of the portfolio for the selected date. | มูลค่าทรัพย์สินสุทธิปัจจุบันของพอร์ต ณ วันที่เลือก |
| Available Unit          | The total number of units currently held in the portfolio.      | จำนวนหน่วยลงทุนทั้งหมดที่มีอยู่ในปัจจุบัน          |
| Subscription Base / (+) | Basis and amount of new unit subscriptions (increase).          | เกณฑ์และจำนวนหน่วยที่มีการซื้อเพิ่ม (Subscription) |
| Redemption Base / (-)   | Basis and amount of unit redemptions (decrease).                | เกณฑ์และจำนวนหน่วยที่มีการขายคืน (Redemption)      |
| Daily Return Base       | The percentage or base used to calculate daily returns.         | เกณฑ์ที่ใช้ในการคำนวณผลตอบแทนรายวัน                |
| Action (Edit/Save)      | Icons to initiate editing or commit changes to the units.       | ไอคอนสำหรับเริ่มแก้ไข หรือบันทึกการเปลี่ยนแปลง     |
