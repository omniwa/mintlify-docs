# Adjust NAV of security

### Adjust NAV of Security

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (330).png" alt=""><figcaption></figcaption></figure></div>

| **Component**            | **Description (English)**                                  | **คำอธิบาย (ภาษาไทย)**                             |
| ------------------------ | ---------------------------------------------------------- | -------------------------------------------------- |
| Portfolio (Filter)       | Select the specific portfolio to view its asset valuation. | เลือกพอร์ตโฟลิโอที่ต้องการตรวจสอบมูลค่าสินทรัพย์   |
| Security (Filter)        | Search for a specific asset within the portfolio.          | ค้นหาหลักทรัพย์รายตัวที่ต้องการตรวจสอบ             |
| Value Date (Filter)      | Specify the effective date for the valuation review.       | เลือกวันที่ต้องการตรวจสอบหรือปรับปรุงมูลค่า        |
| Security Code / Currency | The identifier of the asset and its trading currency.      | รหัสหลักทรัพย์และสกุลเงินที่ใช้ในการซื้อขาย        |
| Market Cost              | The current market value calculated by the system.         | มูลค่าตลาดปัจจุบันที่ระบบคำนวณให้โดยอัตโนมัติ      |
| Adj. Market Cost         | Field to manually adjust the market value of the security. | ช่องสำหรับกรอกเพื่อปรับปรุงมูลค่าตลาดด้วยตนเอง     |
| Accrued Int.             | Calculated accrued interest (for debt or cash assets).     | ดอกเบี้ยค้างรับที่ระบบคำนวณให้ ณ วันที่เลือก       |
| Adj Accrued Int.         | Field to manually adjust the accrued interest amount.      | ช่องสำหรับปรับปรุงยอดดอกเบี้ยค้างรับด้วยตนเอง      |
| Edit / Save / Cancel     | Control buttons to modify and confirm valuation changes.   | ปุ่มสำหรับแก้ไข บันทึก หรือยกเลิกการปรับปรุงข้อมูล |
