# Price Code

### Price Code

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (217).png" alt=""><figcaption></figcaption></figure></div>

| **Component**       | **Description (English)**                          | **คำอธิบาย (ภาษาไทย)**                                             |
| ------------------- | -------------------------------------------------- | ------------------------------------------------------------------ |
| Search Box          | Search for an existing Security Code.              | ช่องสำหรับค้นหารหัสหลักทรัพย์ที่มีอยู่ในระบบ                       |
| Create Button       | Add a new Security Code to the system.             | ปุ่มสำหรับเพิ่มรหัสหลักทรัพย์ใหม่เข้าสู่ระบบ                       |
| Security Code Table | Lists all registered and active Security Codes.    | ตารางแสดงรายการรหัสหลักทรัพย์ที่ลงทะเบียนและใช้งานอยู่ในปัจจุบัน   |
| 👁 Preview Button   | View the full details of a specific Security Code. | ปุ่มสำหรับเรียกดูรายละเอียดทั้งหมดของรหัสหลักทรัพย์นั้น ๆ          |
| 🖊 Edit Button      | Edit or update an existing Security Code.          | ปุ่มสำหรับแก้ไขหรืออัปเดตข้อมูลของรหัสหลักทรัพย์ที่มีอยู่          |
| 🗑️ Delete Button   | Remove a Security Code from the system.            | ปุ่มสำหรับลบรหัสหลักทรัพย์ออกจากระบบ                               |
| Pending Table       | Security Codes awaiting approval or verification.  | ตารางแสดงรายการรหัสหลักทรัพย์ที่อยู่ระหว่างรอการตรวจสอบหรืออนุมัติ |

### Create Price Code

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (218).png" alt=""><figcaption></figcaption></figure></div>

| **Field / Button** | **Description (English)**                          | **คำอธิบาย (ภาษาไทย)**                            |
| ------------------ | -------------------------------------------------- | ------------------------------------------------- |
| Security Code      | Asset identifier.                                  | รหัสสำหรับระบุตัวตนของหลักทรัพย์                  |
| Price Source Code  | Linked Price Source for this security.             | แหล่งข้อมูลราคาที่เชื่อมโยงกับหลักทรัพย์นี้       |
| Code               | Asset classification code.                         | รหัสสำหรับการจัดหมวดหมู่ประเภทของหลักทรัพย์       |
| Add Button         | Add another row for multiple entries.              | ปุ่มสำหรับเพิ่มแถวใหม่ เพื่อกรอกข้อมูลเพิ่มเติม   |
| Create Button      | Save the entry to the system.                      | ปุ่มสำหรับบันทึกข้อมูลเข้าสู่ระบบ                 |
| Cancel Button      | Discard the entry and return to the previous page. | ปุ่มสำหรับยกเลิกการกรอกข้อมูลและกลับไปยังหน้าหลัก |
