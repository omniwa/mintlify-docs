# Exchange Market

### Exchange Market

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (223).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                            | **คำอธิบาย (ภาษาไทย)**                                             |
| --------------------- | ---------------------------------------------------- | ------------------------------------------------------------------ |
| ⌕ Search Box          | Search for an existing Exchange Market record.       | ช่องสำหรับค้นหาข้อมูลตลาดหลักทรัพย์ที่มีอยู่ในระบบ                 |
| Create Button         | Add a new Exchange Market to the system.             | ปุ่มสำหรับเพิ่มข้อมูลตลาดหลักทรัพย์ใหม่เข้าสู่ระบบ                 |
| Exchange Market Table | Lists all registered and active Exchange Markets.    | ตารางแสดงรายการตลาดหลักทรัพย์ที่ลงทะเบียนและใช้งานอยู่ในปัจจุบัน   |
| 👁 Preview Button     | View the full details of a specific Exchange Market. | ปุ่มสำหรับเรียกดูรายละเอียดทั้งหมดของตลาดหลักทรัพย์นั้น ๆ          |
| 🖊 Edit Button        | Edit or update an existing Exchange Market record.   | ปุ่มสำหรับแก้ไขหรืออัปเดตข้อมูลของตลาดหลักทรัพย์ที่มีอยู่          |
| 🗑️ Delete Button     | Remove an Exchange Market from the system.           | ปุ่มสำหรับลบข้อมูลตลาดหลักทรัพย์ออกจากระบบ                         |
| Pending Table         | Exchange Markets awaiting approval or verification.  | ตารางแสดงรายการตลาดหลักทรัพย์ที่อยู่ระหว่างรอการตรวจสอบหรืออนุมัติ |



### Create Exchange Market

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (225).png" alt=""><figcaption></figcaption></figure></div>

| **Field / Button** | **Description (English)**                          | **คำอธิบาย (ภาษาไทย)**                              |
| ------------------ | -------------------------------------------------- | --------------------------------------------------- |
| Exchange Code      | Unique identifier for the exchange.                | รหัสเฉพาะสำหรับระบุตลาดหลักทรัพย์                   |
| Description        | Summary or full name of the exchange.              | ชื่อหรือรายละเอียดโดยย่อของตลาดหลักทรัพย์           |
| Calendar           | Trading and settlement calendar settings.          | การตั้งค่าปฏิทินวันซื้อขายและวันทำการชำระราคา       |
| Country            | Associated country of the exchange.                | ประเทศที่ตลาดหลักทรัพย์นั้นตั้งอยู่                 |
| Settlement Day(s)  | Settlement business-day count (e.g., T+2).         | จำนวนวันทำการสำหรับการชำระราคาและส่งมอบหลักทรัพย์   |
| MTM Type           | Mark-to-market classification method.              | ประเภทหรือวิธีการวัดมูลค่ายุติธรรม (Mark-to-Market) |
| Type               | Exchange market category or segment.               | การจัดหมวดหมู่หรือประเภทของตลาดหลักทรัพย์           |
| Create Button      | Save the current entry to the system.              | ปุ่มสำหรับบันทึกข้อมูลเข้าสู่ระบบ                   |
| Cancel Button      | Discard the entry and return to the previous page. | ปุ่มสำหรับยกเลิกการกรอกข้อมูลและกลับไปยังหน้าหลัก   |
