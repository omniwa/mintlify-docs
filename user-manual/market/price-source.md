# Price Source

### Price Source

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (220).png" alt=""><figcaption></figcaption></figure></div>

| **Component**      | **Description (English)**                         | **คำอธิบาย (ภาษาไทย)**                                              |
| ------------------ | ------------------------------------------------- | ------------------------------------------------------------------- |
| Search Box         | Search for an existing Price Source.              | ช่องสำหรับค้นหาแหล่งข้อมูลราคาที่มีอยู่ในระบบ                       |
| Create Button      | Add a new Price Source to the system.             | ปุ่มสำหรับเพิ่มแหล่งข้อมูลราคาใหม่เข้าสู่ระบบ                       |
| Price Source Table | Lists all registered and active Price Sources.    | ตารางแสดงรายการแหล่งข้อมูลราคาที่ลงทะเบียนและใช้งานอยู่ในปัจจุบัน   |
| 👁 Preview Button  | View the full details of a specific Price Source. | ปุ่มสำหรับเรียกดูรายละเอียดทั้งหมดของแหล่งข้อมูลราคานั้น ๆ          |
| 🖊 Edit Button     | Edit or update an existing Price Source.          | ปุ่มสำหรับแก้ไขหรืออัปเดตข้อมูลของแหล่งข้อมูลราคาที่มีอยู่          |
| 🗑️ Delete Button  | Remove a Price Source from the system.            | ปุ่มสำหรับลบแหล่งข้อมูลราคาออกจากระบบ                               |
| Pending Table      | Price Sources awaiting approval or verification.  | ตารางแสดงรายการแหล่งข้อมูลราคาที่อยู่ระหว่างรอการตรวจสอบหรืออนุมัติ |

### Create Price Source

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (221).png" alt=""><figcaption></figcaption></figure></div>

| **Field / Button** | **Description (English)**                                 | **คำอธิบาย (ภาษาไทย)**                                        |
| ------------------ | --------------------------------------------------------- | ------------------------------------------------------------- |
| Price Source Code  | Unique identifier for the Price Source.                   | รหัสเฉพาะสำหรับระบุแหล่งที่มาของราคา                          |
| Description        | Primary description of the price source.                  | รายละเอียดคำอธิบายหลักของแหล่งที่มาของราคา                    |
| Description2       | Additional or secondary description.                      | รายละเอียดคำอธิบายเพิ่มเติมหรือข้อมูลสำรอง                    |
| Trade With Broker  | Whether the source is a broker that the firm trades with. | ระบุว่าเป็นโบรกเกอร์ที่บริษัทมีการทำธุรกรรมซื้อขายด้วยหรือไม่ |
| Create Button      | Save the current entry to the system.                     | ปุ่มสำหรับบันทึกข้อมูลเข้าสู่ระบบ                             |
| Cancel Button      | Discard the current entry and exit.                       | ปุ่มสำหรับยกเลิกการกรอกข้อมูลและออกจากหน้านี้                 |
