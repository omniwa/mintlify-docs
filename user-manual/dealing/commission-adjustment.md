# Commission Adjustment

### Commission Adjustment





| **Component**         | **Description (English)**                                            | **คำอธิบาย (ภาษาไทย)**                                       |
| --------------------- | -------------------------------------------------------------------- | ------------------------------------------------------------ |
| Search by Broker      | Filter the list to display transactions from a specific broker.      | ค้นหารายการธุรกรรมตามชื่อของโบรกเกอร์                        |
| Allocations           | Options to manage or view related allocation details.                | ตัวเลือกในการจัดการหรือดูรายละเอียดการจัดสรรที่เกี่ยวข้อง    |
| Download Excel        | Export the current commission adjustment data to an Excel file.      | ปุ่มสำหรับส่งออกข้อมูลการปรับปรุงค่าคอมมิชชั่นเป็นไฟล์ Excel |
| Transaction Table     | List of all transactions currently subject to commission adjustment. | ตารางแสดงรายการธุรกรรมทั้งหมดที่รอการปรับปรุงค่าคอมมิชชั่น   |
| Edit Button           | Enable the adjustment field to modify the commission value.          | ปุ่มสำหรับเปิดโหมดแก้ไขเพื่อปรับปรุงยอดค่าคอมมิชชั่น         |
| Adj. Commission Field | Input field for entering the new, corrected commission amount.       | ช่องสำหรับกรอกจำนวนเงินค่าคอมมิชชั่นใหม่ที่ต้องการปรับปรุง   |
| Submit Button         | Commit and save the adjusted commission to the system.               | ปุ่มสำหรับบันทึกการปรับปรุงค่าคอมมิชชั่นเข้าสู่ระบบ          |





### Bulk upload



| **Component**     | **Description (English)**                                   | **คำอธิบาย (ภาษาไทย)**                                       |
| ----------------- | ----------------------------------------------------------- | ------------------------------------------------------------ |
| Upload File       | Open the local file explorer to select a data file.         | ปุ่มเปิดหน้าต่างเลือกไฟล์จากเครื่องคอมพิวเตอร์               |
| Drag & Drop Area  | Area to drag an `.xlsx` file directly for a faster upload.  | พื้นที่สำหรับลากและวางไฟล์ Excel (.xlsx) เพื่อความรวดเร็ว    |
| Upload Button     | Confirm the selected file and initiate the data import.     | ปุ่มยืนยันเพื่อเริ่มกระบวนการนำเข้าข้อมูลเข้าสู่ระบบ         |
| Transaction Table | Display area for the rows and data extracted from the file. | ตารางแสดงรายการข้อมูลที่ถูกดึงมาจากไฟล์เพื่อตรวจสอบเบื้องต้น |
