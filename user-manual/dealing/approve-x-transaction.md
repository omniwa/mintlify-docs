# Approve X-Transaction

### Approve X-Transaction

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (338).png" alt=""><figcaption></figcaption></figure></div>

| **Component**       | **Description (English)**                                        | **คำอธิบาย (ภาษาไทย)**                                                       |
| ------------------- | ---------------------------------------------------------------- | ---------------------------------------------------------------------------- |
| Status Filter       | Filter transactions by All, Unapproved, or Approved status.      | กรองรายการตามสถานะ ทั้งหมด, ยังไม่อนุมัติ หรือ อนุมัติแล้ว                   |
| Status Icon         | Visual indicator (Checkmark/Cross) of the approval state.        | ไอคอนแสดงสถานะ (เครื่องหมายถูกคืออนุมัติแล้ว / กากบาทคือยังไม่อนุมัติ)       |
| X-Type              | Type of Corporate Action (e.g., XD, XR, XE, CONVERT).            | ประเภทของสิทธิประโยชน์ (เช่น ปันผล, สิทธิซื้อหุ้นเพิ่มทุน, ใช้สิทธิ Warrant) |
| Security Code       | The symbol of the underlying asset gaining the right.            | รหัสหลักทรัพย์ที่ได้รับสิทธินั้น ๆ                                           |
| X-Date              | The date on which the security begins trading without the right. | วันที่ขึ้นเครื่องหมาย (วันที่ผู้ซื้อหลักทรัพย์จะไม่ได้สิทธินั้น)             |
| Payment Date        | The official date when the benefit (cash/stock) is distributed.  | วันที่จ่ายเงินปันผลหรือส่งมอบหุ้นเข้าพอร์ต                                   |
| Tradable Date       | The date from which the new securities can be traded.            | วันที่หลักทรัพย์ใหม่ (ถ้ามี) เริ่มซื้อขายในตลาดได้                           |
| Right Security Code | The code for the new asset or right being received.              | รหัสของหลักทรัพย์ใหม่หรือสิทธิที่ได้รับ (เช่น หุ้นแม่-XR)                    |
| Old / New Ratio     | The entitlement ratio for the corporate action (e.g., 1:1).      | อัตราส่วนการให้สิทธิ (จำนวนหุ้นเดิม : จำนวนสิทธิใหม่)                        |
