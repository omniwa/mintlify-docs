# Broker Group

### Broker Group

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (339).png" alt=""><figcaption></figcaption></figure></div>

| **Component**        | **Description (English)**                                   | **คำอธิบาย (ภาษาไทย)**                                     |
| -------------------- | ----------------------------------------------------------- | ---------------------------------------------------------- |
| Create Broker Group  | Button to define and add a new group of brokers.            | ปุ่มสำหรับสร้างและเพิ่มกลุ่มโบรกเกอร์ใหม่                  |
| Name                 | The unique identifier or name of the broker group.          | ชื่อหรือตัวระบุกลุ่มของโบรกเกอร์                           |
| Wiggle Room 1 (%)    | Primary tolerance threshold for transaction reconciliation. | เกณฑ์ความคลาดเคลื่อนลำดับแรกที่ยอมรับได้ (เป็นเปอร์เซ็นต์) |
| Wiggle Room 2 (%)    | Secondary or maximum tolerance threshold.                   | เกณฑ์ความคลาดเคลื่อนลำดับที่สองหรือระดับสูงสุดที่ยอมรับได้ |
| View / Edit / Delete | Icons to inspect, modify settings, or remove the group.     | ไอคอนสำหรับดูรายละเอียด, แก้ไขการตั้งค่า หรือลบกลุ่มทิ้ง   |



### Create Broker Group

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (340).png" alt=""><figcaption></figcaption></figure></div>

| **Component**     | **Description (English)**                                               | **คำอธิบาย (ภาษาไทย)**                                              |
| ----------------- | ----------------------------------------------------------------------- | ------------------------------------------------------------------- |
| Broker Group Name | The unique name used to identify the broker group.                      | ชื่อที่ใช้ระบุกลุ่มของโบรกเกอร์                                     |
| Wiggle Room 1 & 2 | Flexibility percentages for trade matching (currently disabled).        | เปอร์เซ็นต์ความยืดหยุ่นในการจับคู่รายการ (ปัจจุบันยังไม่เปิดใช้งาน) |
| Broker Available  | List of brokers that are not yet assigned to any group.                 | รายชื่อโบรกเกอร์ที่ยังว่างอยู่และสามารถเลือกเข้ากลุ่มได้            |
| Move Button (>)   | Transfer selected brokers from the Available list to the Selected list. | ปุ่มสำหรับย้ายโบรกเกอร์ที่เลือกจากฝั่ง Available ไปยังฝั่ง Selected |
| Broker Selected   | List of brokers currently included in this specific group.              | รายชื่อโบรกเกอร์ที่ถูกเลือกเข้ามาอยู่ในกลุ่มนี้แล้ว                 |
| Create Button     | Save the new group or commit changes to an existing one.                | ปุ่มสำหรับบันทึกการสร้างกลุ่มใหม่หรือบันทึกการแก้ไข                 |
| Cancel Button     | Discard all changes and return to the main Broker Group page.           | ปุ่มยกเลิกการตั้งค่าทั้งหมดและกลับสู่หน้าหลัก                       |
