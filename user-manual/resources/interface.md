# Interface

### Interface

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (149).png" alt=""><figcaption></figcaption></figure></div>

**Upload Files**: Click or drag a .XLSX file to upload data.

**Start Upload**: Begin processing the file.

**Auto Prove**: Automatically verify and process the file.

**Download Example File** (.xlsx): Sample file showing the required format.

**Data Table**: Shows the uploaded data for review.
