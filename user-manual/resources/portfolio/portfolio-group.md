# Portfolio Group

### Portfolio Group

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (279).png" alt=""><figcaption></figcaption></figure></div>

| **Component**                 | **Description (English)**                                      | **คำอธิบาย (ภาษาไทย)**                                                    |
| ----------------------------- | -------------------------------------------------------------- | ------------------------------------------------------------------------- |
| Portfolio Group Code (Search) | Search for an existing portfolio group by its unique code.     | ช่องสำหรับค้นหากลุ่มพอร์ตโฟลิโอที่มีอยู่แล้วด้วยรหัสกลุ่ม                 |
| Create Portfolio Group        | Open the setup to define and add a new portfolio group.        | ปุ่มสำหรับสร้างและกำหนดรายละเอียดกลุ่มพอร์ตโฟลิโอใหม่                     |
| Portfolio Group Table         | Lists all currently configured portfolio groups in the system. | ตารางแสดงรายชื่อกลุ่มพอร์ตโฟลิโอทั้งหมดที่ตั้งค่าไว้ในระบบ                |
| Preview Button                | View detailed information and member portfolios of the group.  | ปุ่มสำหรับเรียกดูรายละเอียดข้อมูลและรายชื่อพอร์ตสมาชิกในกลุ่ม             |
| Edit Button                   | Modify the settings or composition of an existing group.       | ปุ่มสำหรับแก้ไขการตั้งค่าหรือปรับเปลี่ยนสมาชิกในกลุ่มเดิม                 |
| Delete Button                 | Permanently remove the portfolio group from the system.        | ปุ่มสำหรับลบกลุ่มพอร์ตโฟลิโอออกจากระบบ                                    |
| Pending Portfolio Group Table | Groups currently awaiting final approval or completion.        | ตารางแสดงกลุ่มพอร์ตโฟลิโอที่อยู่ระหว่างรอการอนุมัติหรือยังไม่เสร็จสมบูรณ์ |



### Create Portfolio Group

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (280).png" alt=""><figcaption></figcaption></figure></div>

| **Component**        | **Description (English)**                                        | **คำอธิบาย (ภาษาไทย)**                                                |
| -------------------- | ---------------------------------------------------------------- | --------------------------------------------------------------------- |
| Name                 | The designated name for the portfolio group.                     | ชื่อของกลุ่มพอร์ตโฟลิโอ                                               |
| Portfolio Group Code | A unique identifier code for the group.                          | รหัสเฉพาะสำหรับระบุกลุ่มพอร์ตโฟลิโอ                                   |
| Description          | Additional notes or details regarding the group's purpose.       | ข้อมูลเพิ่มเติมหรือรายละเอียดเกี่ยวกับวัตถุประสงค์ของกลุ่ม            |
| Portfolio Available  | A list of existing portfolios that can be added to this group.   | รายชื่อพอร์ตโฟลิโอทั้งหมดที่มีอยู่และสามารถเลือกเข้ากลุ่มได้          |
| Move Button ( > )    | Transfer the selected portfolios from "Available" to "Selected". | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกจากช่อง Available ไปยังช่อง Selected |
| Portfolio Selected   | The specific portfolios that have been assigned to this group.   | รายชื่อพอร์ตโฟลิโอที่ถูกเลือกให้เป็นสมาชิกในกลุ่มนี้                  |
| Create Button        | Save the configuration and create the portfolio group.           | ปุ่มสำหรับบันทึกการตั้งค่าและสร้างกลุ่มพอร์ตโฟลิโอ                    |
| Cancel Button        | Discard all current input and return to the previous page.       | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้ทั้งหมดและกลับสู่หน้าก่อนหน้า           |



### Don't forget to approve the actions in the Pending Table to finalize the process.
