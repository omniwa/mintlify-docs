# Portfolio Model

### Portfolio Model

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (281).png" alt=""><figcaption></figcaption></figure></div>

| **Component**                 | **Description (English)**                                      | **คำอธิบาย (ภาษาไทย)**                                              |
| ----------------------------- | -------------------------------------------------------------- | ------------------------------------------------------------------- |
| Portfolio Model Code (Search) | Search for an existing portfolio model by its unique code.     | ช่องสำหรับค้นหาโมเดลพอร์ตโฟลิโอที่มีอยู่แล้วด้วยรหัสโมเดล           |
| Create Portfolio Model        | Open the setup to define and add a new portfolio model.        | ปุ่มสำหรับเริ่มสร้างและกำหนดรายละเอียดโมเดลพอร์ตโฟลิโอใหม่          |
| Portfolio Model Table         | Lists all currently configured portfolio models in the system. | ตารางแสดงรายชื่อโมเดลพอร์ตโฟลิโอทั้งหมดที่ตั้งค่าไว้ในระบบ          |
| Preview Button                | View the structure and asset allocation details of the model.  | ปุ่มสำหรับเรียกดูโครงสร้างและรายละเอียดการจัดสรรสินทรัพย์ของโมเดล   |
| Edit Button                   | Modify the settings or asset composition of an existing model. | ปุ่มสำหรับแก้ไขการตั้งค่าหรือปรับเปลี่ยนสัดส่วนสินทรัพย์ในโมเดลเดิม |
| Delete Button                 | Remove the portfolio model from the system.                    | ปุ่มสำหรับลบโมเดลพอร์ตโฟลิโอออกจากระบบ                              |
| Pending Portfolio Model Table | Models currently awaiting final approval before activation.    | ตารางแสดงโมเดลพอร์ตโฟลิโอที่อยู่ระหว่างรอการอนุมัติเพื่อเริ่มใช้งาน |



### Create Portfolio Model

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (282).png" alt=""><figcaption></figcaption></figure></div>

| **Component**       | **Description (English)**                                           | **คำอธิบาย (ภาษาไทย)**                                                  |
| ------------------- | ------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| Portfolio Model     | Select the specific investment model to be assigned.                | เลือกโมเดลการลงทุนที่ต้องการนำไปใช้งานกับพอร์ต                          |
| Portfolio Available | A list of portfolios that have not yet been assigned to this model. | รายชื่อพอร์ตโฟลิโอที่ว่างอยู่และสามารถเลือกเข้าโมเดลได้                 |
| Move Button ( > )   | Assign the selected portfolios to the chosen model.                 | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังรายการที่ได้รับมอบหมาย (Selected) |
| Portfolio Selected  | Portfolios currently assigned and following this investment model.  | รายชื่อพอร์ตโฟลิโอที่ถูกกำหนดให้ใช้โมเดลการลงทุนนี้แล้ว                 |
| Create Button       | Save the assignment and link the portfolios to the model.           | ปุ่มสำหรับบันทึกการมอบหมายโมเดลให้กับพอร์ตโฟลิโอ                        |
| Cancel Button       | Discard the current selection and close the window.                 | ปุ่มสำหรับยกเลิกการตั้งค่าและปิดหน้าต่าง                                |



### Don't forget to approve the actions in the Pending Table to finalize the process.
