# Fee Account Code

### Fee Account Code

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (285).png" alt=""><figcaption></figcaption></figure></div>

| **Component**                  | **Description (English)**                                             | **คำอธิบาย (ภาษาไทย)**                                          |
| ------------------------------ | --------------------------------------------------------------------- | --------------------------------------------------------------- |
| Fee GL Code (Search)           | Search for an existing General Ledger (GL) code by its identifier.    | ช่องสำหรับค้นหารหัสบัญชีแยกประเภท (GL) ที่มีอยู่แล้วในระบบ      |
| Create Fee Account Code        | Define and register a new fee-related account code.                   | ปุ่มสำหรับเริ่มสร้างและลงทะเบียนรหัสบัญชีค่าธรรมเนียมใหม่       |
| Fee Account Code Table         | Lists all currently configured fee account codes in the system.       | ตารางแสดงรายชื่อรหัสบัญชีค่าธรรมเนียมทั้งหมดที่ตั้งค่าไว้ในระบบ |
| Preview Button                 | View the specific properties and linkage of the account code.         | ปุ่มสำหรับเรียกดูรายละเอียดและการเชื่อมโยงของรหัสบัญชีนั้น ๆ    |
| Edit Button                    | Modify the details or mappings of an existing account code.           | ปุ่มสำหรับแก้ไขรายละเอียดหรือการจับคู่ข้อมูลของรหัสบัญชีเดิม    |
| Delete Button                  | Remove an account code configuration from the system.                 | ปุ่มสำหรับลบรหัสบัญชีออกจากระบบ                                 |
| Pending Fee Account Code Table | Account codes that are newly created or edited and awaiting approval. | ตารางแสดงรหัสบัญชีที่อยู่ระหว่างรอการอนุมัติก่อนเริ่มใช้งาน     |



### Create Fee Account Code

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (286).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                               | **คำอธิบาย (ภาษาไทย)**                                            |
| --------------------- | ------------------------------------------------------- | ----------------------------------------------------------------- |
| Fee GL Code           | The unique identifier for the General Ledger code.      | รหัสบัญชีแยกประเภทเฉพาะสำหรับระบุรายการนี้                        |
| A/P — Fee & Expense   | Accounts Payable (Liability) for fees and expenses.     | บัญชีเจ้าหนี้การค้าสำหรับค่าธรรมเนียมและค่าใช้จ่าย (หนี้สิน)      |
| A/P — VAT Amount      | Accounts Payable account specifically for VAT.          | บัญชีเจ้าหนี้การค้าสำหรับภาษีมูลค่าเพิ่ม                          |
| Prepaid Fee / Expense | Asset account for advance payments of fees or expenses. | บัญชีค่าธรรมเนียมหรือค่าใช้จ่ายจ่ายล่วงหน้า (สินทรัพย์)           |
| Fee / Expense         | Standard account classification for the fee/expense.    | บัญชีหมวดค่าธรรมเนียมหรือค่าใช้จ่ายตามปกติ                        |
| VAT Amount            | Dedicated account for tracking VAT amounts.             | บัญชีสำหรับบันทึกจำนวนภาษีมูลค่าเพิ่ม                             |
| Create Button         | Save and register the new Fee GL Code.                  | ปุ่มสำหรับบันทึกและลงทะเบียนรหัสบัญชีใหม่                         |
| Cancel Button         | Discard all current input and close the setup.          | <p></p><p></p><p>ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้และปิดหน้าต่าง</p> |

### Don't forget to approve the action in the Pending Table to finalize the creation of your new Fee Account Code.
