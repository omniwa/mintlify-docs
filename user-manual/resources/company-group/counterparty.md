# Counterparty

### Counterparty

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (306).png" alt=""><figcaption></figcaption></figure></div>

| **Component**              | **Description (English)**                                                             | **คำอธิบาย (ภาษาไทย)**                                      |
| -------------------------- | ------------------------------------------------------------------------------------- | ----------------------------------------------------------- |
| Counterparty Code (Search) | Search for an existing counterparty using their unique identification code.           | ช่องสำหรับค้นหาคู่สัญญาที่มีอยู่แล้วด้วยรหัสคู่สัญญา        |
| Create Counterparty        | Open the setup to register and add a new counterparty to the system.                  | ปุ่มสำหรับเริ่มลงทะเบียนและเพิ่มข้อมูลคู่สัญญาใหม่          |
| Counterparty Table         | Lists all counterparties currently registered and active in the system.               | ตารางแสดงรายชื่อคู่สัญญาหลักทั้งหมดที่บันทึกไว้ในระบบ       |
| Preview Button             | View the detailed profile, legal status, and settings of the counterparty.            | ปุ่มสำหรับเรียกดูโปรไฟล์และรายละเอียดเชิงลึกของคู่สัญญา     |
| Edit Button                | Modify the details or configuration of an existing counterparty profile.              | ปุ่มสำหรับแก้ไขข้อมูลหรือการตั้งค่าของคู่สัญญาเดิมที่มีอยู่ |
| Delete Button              | Remove the counterparty record from the system database.                              | ปุ่มสำหรับลบข้อมูลคู่สัญญาออกจากระบบ                        |
| Pending Counterparty Table | Counterparties that have been recently created or modified and are awaiting approval. | ตารางแสดงรายชื่อคู่สัญญาที่รอการอนุมัติก่อนเริ่มใช้งานจริง  |



### Create Counterparty

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (307).png" alt=""><figcaption></figcaption></figure></div>

| **Component**        | **Description (English)**                                                            | **คำอธิบาย (ภาษาไทย)**                                        |
| -------------------- | ------------------------------------------------------------------------------------ | ------------------------------------------------------------- |
| Counterparty Code    | A unique identifier code for the counterparty within the system.                     | รหัสเฉพาะสำหรับระบุตัวตนของคู่สัญญาในระบบ                     |
| Description          | A brief summary or additional details about the counterparty.                        | รายละเอียดสังเขปหรือข้อมูลเพิ่มเติมเกี่ยวกับคู่สัญญานี้       |
| Reference Code       | An additional code used for tracking or mapping with external systems.               | รหัสอ้างอิงเพิ่มเติมสำหรับใช้ติดตามหรือเชื่อมโยงกับระบบภายนอก |
| Counterparty Name    | The official legal name of the counterparty.                                         | ชื่ออย่างเป็นทางการของคู่สัญญา                                |
| Individual / Company | Specify whether the counterparty is an individual person or a business/organization. | ระบุประเภทของคู่สัญญา (บุคคลธรรมดา หรือ นิติบุคคล)            |
| Create Button        | Save the configuration and register the new counterparty.                            | ปุ่มสำหรับบันทึกข้อมูลและลงทะเบียนคู่สัญญาใหม่                |
| Cancel Button        | Discard all current input and return to the management page.                         | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้ทั้งหมดและกลับสู่หน้ารายการหลัก |

### Don't forget to approve the action in the Pending Table to finalize the registration of your new counterparty.
