# Bank

### Bank

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (310).png" alt=""><figcaption></figcaption></figure></div>

| **Component**            | **Description (English)**                                                | **คำอธิบาย (ภาษาไทย)**                                        |
| ------------------------ | ------------------------------------------------------------------------ | ------------------------------------------------------------- |
| Bank Code (Search)       | Search for an existing bank using its unique identifier code.            | ช่องสำหรับค้นหาธนาคารที่มีอยู่ในระบบด้วยรหัสธนาคาร            |
| Create Bank              | Open the interface to register and add a new bank to the system.         | ปุ่มสำหรับเริ่มลงทะเบียนและเพิ่มข้อมูลธนาคารรายใหม่           |
| Bank Group Table         | A comprehensive list of all banks currently registered in the system.    | ตารางแสดงรายชื่อธนาคารทั้งหมดที่บันทึกไว้ในระบบ               |
| Preview Button           | View the complete profile and configuration details of a specific bank.  | ปุ่มสำหรับเรียกดูรายละเอียดและข้อมูลการตั้งค่าของธนาคารนั้น ๆ |
| Edit Button              | Update or modify the information of an existing bank entry.              | ปุ่มสำหรับแก้ไขข้อมูลหรือปรับปรุงรายละเอียดของธนาคารเดิม      |
| Delete Button            | Remove a bank's record from the system database.                         | ปุ่มสำหรับลบข้อมูลธนาคารออกจากระบบ                            |
| Pending Bank Group Table | Banks that have been recently added or edited and are awaiting approval. | ตารางแสดงรายชื่อธนาคารที่รอการอนุมัติก่อนเริ่มใช้งานจริง      |



### Create Bank

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (311).png" alt=""><figcaption></figcaption></figure></div>

| **Component** | **Description (English)**                                            | **คำอธิบาย (ภาษาไทย)**                                        |
| ------------- | -------------------------------------------------------------------- | ------------------------------------------------------------- |
| Bank Code     | A unique identifier code for the bank within the system.             | รหัสเฉพาะสำหรับระบุธนาคารในระบบ                               |
| Bank Name     | The official legal name of the banking institution.                  | ชื่อธนาคารอย่างเป็นทางการ                                     |
| Company Code  | Select the registered company associated with this bank.             | เลือกบริษัทที่ลงทะเบียนไว้เพื่อเชื่อมโยงกับธนาคารนี้          |
| Contact Name  | The name of the primary contact person or relationship manager.      | ชื่อผู้ติดต่อประสานงานหลักของทางธนาคาร                        |
| Contact Phone | The primary phone number for reaching the bank representative.       | หมายเลขโทรศัพท์สำหรับติดต่อเจ้าหน้าที่ธนาคาร                  |
| Email Address | The official contact email for financial inquiries or confirmations. | ที่อยู่อีเมลสำหรับการติดต่อหรือรับส่งเอกสารยืนยันรายการ       |
| Create Button | Save the configuration and register the new bank in the system.      | ปุ่มสำหรับบันทึกข้อมูลและลงทะเบียนธนาคารใหม่                  |
| Cancel Button | Discard all current input and return to the bank management page.    | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้ทั้งหมดและกลับสู่หน้ารายการหลัก |



### Don't forget to approve the action in the Pending Table to finalize the registration of your new bank.
