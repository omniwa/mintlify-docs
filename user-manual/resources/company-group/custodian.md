# Custodian

### Custodian

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (308).png" alt=""><figcaption></figcaption></figure></div>

| **Component**           | **Description (English)**                                                         | **คำอธิบาย (ภาษาไทย)**                                              |
| ----------------------- | --------------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| Custodian Code (Search) | Search for an existing custodian using their unique identification code.          | ช่องสำหรับค้นหาผู้ดูแลรักษาทรัพย์สินที่มีอยู่แล้วด้วยรหัสคัสโตเดียน |
| Create Custodian        | Open the setup to register and add a new custodian to the system.                 | ปุ่มสำหรับเริ่มลงทะเบียนและเพิ่มข้อมูลผู้ดูแลรักษาทรัพย์สินรายใหม่  |
| Custodian Table         | Lists all custodians currently registered and active in the system.               | ตารางแสดงรายชื่อคัสโตเดียนทั้งหมดที่บันทึกไว้ในระบบ                 |
| Preview Button          | View the detailed profile and associated accounts of the custodian.               | ปุ่มสำหรับเรียกดูโปรไฟล์และข้อมูลโดยละเอียดของผู้ดูแลรักษาทรัพย์สิน |
| Edit Button             | Modify the details or configuration of an existing custodian profile.             | ปุ่มสำหรับแก้ไขข้อมูลหรือการตั้งค่าของคัสโตเดียนเดิมที่มีอยู่       |
| Delete Button           | Remove the custodian record from the system database.                             | ปุ่มสำหรับลบข้อมูลคัสโตเดียนออกจากระบบ                              |
| Pending Custodian Table | Custodians that have been recently created or modified and are awaiting approval. | ตารางแสดงรายชื่อคัสโตเดียนที่รอการอนุมัติก่อนเริ่มใช้งานจริง        |



### Create Custodian

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (309).png" alt=""><figcaption></figcaption></figure></div>

| **Component**  | **Description (English)**                                          | **คำอธิบาย (ภาษาไทย)**                                        |
| -------------- | ------------------------------------------------------------------ | ------------------------------------------------------------- |
| Custodian Code | A unique identifier code for the custodian within the system.      | รหัสเฉพาะสำหรับระบุตัวตนของคัสโตเดียนในระบบ                   |
| Custodian Name | The official legal name of the custodian institution.              | ชื่อสถาบันผู้ดูแลรักษาทรัพย์สินอย่างเป็นทางการ                |
| Company Code   | Select the registered company associated with this custodian.      | เลือกบริษัทที่ลงทะเบียนไว้เพื่อเชื่อมโยงกับคัสโตเดียนนี้      |
| Contact Person | The name of the primary officer or contact at the custodian.       | ชื่อเจ้าหน้าที่หรือผู้ติดต่อประสานงานหลักของคัสโตเดียน        |
| Contact Phone  | The primary phone number for reaching the custodian.               | หมายเลขโทรศัพท์สำหรับติดต่อคัสโตเดียน                         |
| Email Address  | The official contact email for settlement instructions or reports. | ที่อยู่อีเมลสำหรับการติดต่อหรือส่งเอกสารแจ้งการชำระราคา       |
| Create Button  | Save the profile and register the new custodian in the system.     | ปุ่มสำหรับบันทึกข้อมูลและลงทะเบียนคัสโตเดียนใหม่              |
| Cancel Button  | Discard all current input and return to the management page.       | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้ทั้งหมดและกลับสู่หน้ารายการหลัก |

### Don't forget to approve the action in the Pending Table to finalize the registration of your new custodian.
