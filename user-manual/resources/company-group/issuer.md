# Issuer

### Issuer

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (289).png" alt=""><figcaption></figcaption></figure></div>

| **Component**        | **Description (English)**                                                      | **คำอธิบาย (ภาษาไทย)**                                                |
| -------------------- | ------------------------------------------------------------------------------ | --------------------------------------------------------------------- |
| Issuer Code (Search) | Search for an existing issuer using their unique identification code.          | ช่องสำหรับค้นหาผู้ออกตราสาร (Issuer) ที่มีอยู่แล้วด้วยรหัสผู้ออก      |
| Create Issuer        | Open the setup to register and add a new issuer to the system.                 | ปุ่มสำหรับเริ่มลงทะเบียนและเพิ่มข้อมูลผู้ออกตราสารรายใหม่             |
| Issuer Table         | Lists all issuers currently configured in the system.                          | ตารางแสดงรายชื่อผู้ออกตราสารทั้งหมดที่ตั้งค่าไว้ในระบบ                |
| Preview Button       | View the detailed profile and associated information of the issuer.            | ปุ่มสำหรับเรียกดูข้อมูลโดยละเอียดและโปรไฟล์ของผู้ออกตราสาร            |
| Edit Button          | Modify the details or settings of an existing issuer profile.                  | ปุ่มสำหรับแก้ไขข้อมูลหรือการตั้งค่าของผู้ออกตราสารเดิม                |
| Delete Button        | Remove the issuer's record from the system database.                           | ปุ่มสำหรับลบข้อมูลผู้ออกตราสารออกจากระบบ                              |
| Pending Issuer Table | Issuers that have been recently created or modified and are awaiting approval. | ตารางแสดงรายชื่อผู้ออกตราสารที่อยู่ระหว่างรอการอนุมัติก่อนเริ่มใช้งาน |



### Create Issuer

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (290).png" alt=""><figcaption></figcaption></figure></div>

| **Component**        | **Description (English)**                                              | **คำอธิบาย (ภาษาไทย)**                                      |
| -------------------- | ---------------------------------------------------------------------- | ----------------------------------------------------------- |
| Company Code         | Select the registered company associated with this issuer.             | เลือกบริษัทที่ลงทะเบียนไว้เพื่อเชื่อมโยงกับผู้ออกตราสารนี้  |
| Individual / Company | Specify whether the issuer is an individual person or an organization. | ระบุประเภทของผู้ออกตราสาร (บุคคลธรรมดา หรือ นิติบุคคล)      |
| Issuer Code          | A unique identifier code for the issuer within the system.             | รหัสเฉพาะสำหรับใช้ระบุตัวตนของผู้ออกตราสารในระบบ            |
| Issuer Name          | The official legal name of the issuer.                                 | ชื่ออย่างเป็นทางการของผู้ออกตราสาร                          |
| Description          | Additional details or notes regarding the issuer.                      | ข้อมูลเพิ่มเติมหรือรายละเอียดสังเขปเกี่ยวกับผู้ออกตราสาร    |
| Create Button        | Save the configuration and register the new issuer.                    | ปุ่มสำหรับบันทึกข้อมูลและลงทะเบียนผู้ออกตราสารใหม่          |
| Cancel Button        | Discard all current input and return to the previous page.             | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้ทั้งหมดและกลับสู่หน้าก่อนหน้า |

### Don't forget to approve the action in the Pending Table to finalize the registration of your new issuer.
