# Business Sector

### Business Sector

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (300).png" alt=""><figcaption></figcaption></figure></div>

| **Component**                 | **Description (English)**                                                    | **คำอธิบาย (ภาษาไทย)**                                        |
| ----------------------------- | ---------------------------------------------------------------------------- | ------------------------------------------------------------- |
| Business Sector Code (Search) | Search for an existing sector using its unique code or name.                 | ช่องสำหรับค้นหากลุ่มธุรกิจที่มีอยู่แล้วในระบบด้วยรหัสหรือชื่อ |
| Create Business Sector        | Define and register a new business sector category.                          | ปุ่มสำหรับเริ่มสร้างและลงทะเบียนหมวดหมู่กลุ่มธุรกิจใหม่       |
| Business Sector Table         | Lists all business sectors currently configured in the system.               | ตารางแสดงรายชื่อกลุ่มธุรกิจทั้งหมดที่ตั้งค่าไว้ในระบบ         |
| Preview Button                | View the specific details and attributes of the sector.                      | ปุ่มสำหรับเรียกดูรายละเอียดและคุณลักษณะของกลุ่มธุรกิจนั้น ๆ   |
| Edit Button                   | Modify the name or properties of an existing sector.                         | ปุ่มสำหรับแก้ไขชื่อหรือข้อมูลของกลุ่มธุรกิจเดิมที่มีอยู่      |
| Delete Button                 | Remove a business sector classification from the database.                   | ปุ่มสำหรับลบหมวดหมู่กลุ่มธุรกิจออกจากระบบ                     |
| Pending Business Sector Table | Sectors that have been recently created or edited and are awaiting approval. | ตารางแสดงรายชื่อกลุ่มธุรกิจที่รอการอนุมัติก่อนเริ่มใช้งานจริง |

### Create Business Sector

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (301).png" alt=""><figcaption></figcaption></figure></div>

| **Component**        | **Description (English)**                                                     | **คำอธิบาย (ภาษาไทย)**                                            |
| -------------------- | ----------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| Business Sector Code | A unique identifier code for the business sector.                             | รหัสเฉพาะสำหรับระบุกลุ่มธุรกิจในระบบ                              |
| Name / Detail        | The official sector name and an alternative name (e.g., in another language). | ชื่อกลุ่มธุรกิจ (ชื่อหลัก) และชื่อสำรอง (เช่น ภาษาอังกฤษ/ภาษาไทย) |
| Create Button        | Save the configuration and register the new business sector.                  | ปุ่มสำหรับบันทึกข้อมูลและลงทะเบียนกลุ่มธุรกิจใหม่                 |
| Cancel Button        | Discard all current input and return to the previous page.                    | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้และกลับสู่หน้าก่อนหน้า              |

### Don't forget to approve the actions in the Pending Table to finalize the creation or updates of your business sectors.
