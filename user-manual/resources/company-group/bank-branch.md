# Bank Branch

### Bank Branch

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (312).png" alt=""><figcaption></figcaption></figure></div>

| **Component**             | **Description (English)**                                                       | **คำอธิบาย (ภาษาไทย)**                                       |
| ------------------------- | ------------------------------------------------------------------------------- | ------------------------------------------------------------ |
| Code (Search)             | Search for a specific branch using the associated equity or internal code.      | ช่องสำหรับค้นหาสาขาธนาคารด้วยรหัสหลักทรัพย์หรือรหัสอ้างอิง   |
| Bank Branch No (Search)   | Search for a branch by its official bank branch identification number.          | ช่องสำหรับค้นหาสาขาธนาคารด้วยหมายเลขรหัสสาขา                 |
| Create Bank Branch        | Open the setup to register and add a new bank branch to the system.             | ปุ่มสำหรับเริ่มลงทะเบียนและเพิ่มข้อมูลสาขาธนาคารใหม่         |
| Bank Branch Table         | Lists all bank branches currently registered and active in the system.          | ตารางแสดงรายชื่อสาขาธนาคารทั้งหมดที่บันทึกไว้ในระบบ          |
| Preview Button            | View the detailed information and contact settings of the branch.               | ปุ่มสำหรับเรียกดูรายละเอียดและข้อมูลเชิงลึกของสาขานั้น ๆ     |
| Edit Button               | Modify the details or configuration of an existing bank branch.                 | ปุ่มสำหรับแก้ไขข้อมูลหรือการตั้งค่าของสาขาเดิมที่มีอยู่      |
| Delete Button             | Remove the bank branch record from the system database.                         | ปุ่มสำหรับลบข้อมูลสาขาธนาคารออกจากระบบ                       |
| Pending Bank Branch Table | Branches that have been recently created or modified and are awaiting approval. | ตารางแสดงรายชื่อสาขาธนาคารที่รอการอนุมัติก่อนเริ่มใช้งานจริง |



### Create Bank Branch

<figure><img src="../../.gitbook/assets/image (313).png" alt=""><figcaption></figcaption></figure>

| **Component**    | **Description (English)**                                    | **คำอธิบาย (ภาษาไทย)**                                        |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------- |
| Bank Branch No   | A unique identification number for the specific bank branch. | หมายเลขรหัสสาขาธนาคารที่ระบุตัวตนเฉพาะของสาขานั้น ๆ           |
| Bank Branch Name | The official name of the bank branch.                        | ชื่อสาขาธนาคารอย่างเป็นทางการ                                 |
| Bank Code        | Select the parent bank associated with this branch.          | เลือกธนาคารหลักที่สาขานี้สังกัดอยู่                           |
| Bank Address     | The physical location or postal address of the branch.       | ที่อยู่ตั้งของสาขาธนาคาร                                      |
| Create Button    | Save the profile and register the new branch in the system.  | ปุ่มสำหรับบันทึกข้อมูลและลงทะเบียนสาขาธนาคารใหม่              |
| Cancel Button    | Discard all current input and return to the management page. | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้ทั้งหมดและกลับสู่หน้ารายการหลัก |

### Don't forget to approve the action in the Pending Table to finalize the registration of your new bank branch.
