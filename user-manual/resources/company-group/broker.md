# Broker

### Broker

Broker adds a broker into the system, enabling transaction processing and other broker-related workflows.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (302).png" alt=""><figcaption></figcaption></figure></div>

| **Component**        | **Description (English)**                                                      | **คำอธิบาย (ภาษาไทย)**                                        |
| -------------------- | ------------------------------------------------------------------------------ | ------------------------------------------------------------- |
| Broker Code (Search) | Search for an existing broker using their unique identification code.          | ช่องสำหรับค้นหาบริษัทหลักทรัพย์ที่มีอยู่แล้วด้วยรหัสโบรกเกอร์ |
| Create Broker        | Open the setup to register and add a new broker to the system.                 | ปุ่มสำหรับเริ่มลงทะเบียนและเพิ่มข้อมูลบริษัทหลักทรัพย์รายใหม่ |
| Broker Table         | Lists all brokers currently registered and active in the system.               | ตารางแสดงรายชื่อโบรกเกอร์ทั้งหมดที่บันทึกไว้ในระบบ            |
| Preview Button       | View the detailed profile, contact info, and settings of the broker.           | ปุ่มสำหรับเรียกดูโปรไฟล์และรายละเอียดการติดต่อของโบรกเกอร์    |
| Edit Button          | Modify the details or configuration of an existing broker profile.             | ปุ่มสำหรับแก้ไขข้อมูลหรือการตั้งค่าของโบรกเกอร์เดิมที่มีอยู่  |
| Delete Button        | Remove the broker's record from the system database.                           | ปุ่มสำหรับลบข้อมูลโบรกเกอร์ออกจากระบบ                         |
| Pending Broker Table | Brokers that have been recently created or modified and are awaiting approval. | ตารางแสดงรายชื่อโบรกเกอร์ที่รอการอนุมัติก่อนเริ่มใช้งานจริง   |



### Create Broker

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (303).png" alt=""><figcaption></figcaption></figure></div>

| **Component** | **Description (English)**                                        | **คำอธิบาย (ภาษาไทย)**                                  |
| ------------- | ---------------------------------------------------------------- | ------------------------------------------------------- |
| Broker Code   | A unique identification code for the broker within the system.   | รหัสเฉพาะสำหรับระบุโบรกเกอร์ในระบบ                      |
| Broker Name   | The official legal name of the brokerage firm.                   | ชื่อบริษัทหลักทรัพย์อย่างเป็นทางการ                     |
| Company Code  | Select the registered company associated with this broker.       | เลือกบริษัทที่ลงทะเบียนไว้เพื่อเชื่อมโยงกับโบรกเกอร์นี้ |
| Description   | A brief description or additional notes about the broker.        | รายละเอียดสังเขปหรือหมายเหตุเพิ่มเติมเกี่ยวกับโบรกเกอร์ |
| Contact Name  | The name of the primary contact person at the brokerage.         | ชื่อผู้ติดต่อประสานงานหลักของบริษัทหลักทรัพย์           |
| Contact Phone | The primary phone number for reaching the broker.                | หมายเลขโทรศัพท์สำหรับติดต่อโบรกเกอร์                    |
| Email Address | The official contact email for trade confirmations or inquiries. | ที่อยู่อีเมลสำหรับการติดต่อหรือส่งเอกสารยืนยันรายการ    |
| Create Button | Save the profile and register the new broker in the system.      | ปุ่มสำหรับบันทึกข้อมูลและลงทะเบียนโบรกเกอร์ใหม่         |
| Cancel Button | Discard all current input and close the setup window.            | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้ทั้งหมดและปิดหน้าต่าง     |



### Don't forget to approve the action in the Pending Table to finalize the registration of your new broker.
