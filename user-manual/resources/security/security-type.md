# Security Type

### Security Type

{% tabs %}
{% tab title="Security Type" %}
### Security Type

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (30).png" alt=""><figcaption></figcaption></figure></div>

***



| **Component**               | **Description (English)**                                                | **คำอธิบาย (ภาษาไทย)**                                          |
| --------------------------- | ------------------------------------------------------------------------ | --------------------------------------------------------------- |
| Security Type Code (Search) | Search for a specific security category using its code (e.g., Bond Gov). | ช่องสำหรับค้นหาประเภทหลักทรัพย์ด้วยรหัส (เช่น Bond Gov)         |
| Create Security Type        | Open the setup to define a new asset category in the system.             | ปุ่มสำหรับเริ่มสร้างหมวดหมู่หลักทรัพย์ใหม่เข้าสู่ระบบ           |
| Security Type Table         | Lists all registered security types with their structural details.       | ตารางแสดงรายชื่อประเภทหลักทรัพย์ทั้งหมดพร้อมรายละเอียดโครงสร้าง |
| Institute                   | The type of issuing institution (e.g., Government, Corporate).           | ประเภทของสถาบันผู้ออกหลักทรัพย์ (เช่น รัฐบาล, เอกชน)            |
| Asset Class                 | The broad category of the asset (e.g., BOND, CASH, EQUITY).              | กลุ่มสินทรัพย์หลัก (เช่น ตราสารหนี้, เงินสด, หุ้นทุน)           |
| Investment                  | Specific investment sub-type or strategy associated with the code.       | ประเภทย่อยของการลงทุนหรือกลยุทธ์ที่ผูกกับรหัสนั้น ๆ             |
| Internal Code               | Internal mapping code for accounting or external system integration.     | รหัสอ้างอิงภายในสำหรับการลงบัญชีหรือเชื่อมโยงระบบอื่น           |
| Ccy (Currency)              | The default currency associated with this security type.                 | สกุลเงินหลักที่ใช้สำหรับหลักทรัพย์ประเภทนี้                     |
| Preview / Edit / Delete     | Standard action buttons to view, modify, or remove records.              | ปุ่มสำหรับเรียกดู แก้ไข หรือลบรายการข้อมูล                      |
| Pending Security Type Table | Security types awaiting approval before being active in the system.      | ตารางแสดงรายการที่รอการอนุมัติก่อนเริ่มใช้งานจริง               |
{% endtab %}

{% tab title="Create" %}
### Create Security Type

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (31).png" alt=""><figcaption></figcaption></figure></div>

***



| **Component**      | **Description (English)**                                                 | **คำอธิบาย (ภาษาไทย)**                                         |
| ------------------ | ------------------------------------------------------------------------- | -------------------------------------------------------------- |
| Security Type Code | A unique identification code for the security category (e.g., CORP-BOND). | รหัสเฉพาะสำหรับระบุประเภทหรือหมวดหมู่ของหลักทรัพย์             |
| Institute          | Select the type of issuing institution associated with this asset.        | เลือกประเภทสถาบันที่ออกหลักทรัพย์ (เช่น Government, Corporate) |
| Asset Class        | Define the broad asset category (e.g., BOND, EQUITY, CASH).               | กำหนดกลุ่มสินทรัพย์หลักเพื่อใช้ในการจัดพอร์ต                   |
| Investment Code    | Internal or reference code used for specific investment strategies.       | รหัสอ้างอิงสำหรับการลงทุนหรือรหัสภายในที่เกี่ยวข้อง            |
| Currency           | Select the primary currency associated with this security type.           | เลือกสกุลเงินหลักที่ใช้สำหรับหลักทรัพย์ประเภทนี้               |
| Create Button      | Save the configuration and register the security type in the system.      | ปุ่มสำหรับบันทึกข้อมูลและลงทะเบียนประเภทหลักทรัพย์ใหม่         |
| Cancel Button      | Discard the current input and return to the management page.              | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้และกลับสู่หน้ารายการหลัก         |



### Don't forget to approve the action in the Pending Table to finalize the creation of your new security type.
