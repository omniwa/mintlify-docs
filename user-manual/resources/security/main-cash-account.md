# Main Cash Account

### Main Cash Account

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (39).png" alt=""><figcaption></figcaption></figure></div>

| **Component**                   | **Description (English)**                                              | **คำอธิบาย (ภาษาไทย)**                                              |
| ------------------------------- | ---------------------------------------------------------------------- | ------------------------------------------------------------------- |
| Portfolio Code (Search)         | Search for specific main cash accounts by their linked portfolio code. | ช่องสำหรับค้นหาบัญชีเงินสดหลักตามรหัสพอร์ตโฟลิโอ                    |
| Create Main Cash Account        | Open the setup to link a default cash security to a portfolio.         | ปุ่มสำหรับเริ่มตั้งค่าเพื่อผูกบัญชีเงินสดเริ่มต้นเข้ากับพอร์ตโฟลิโอ |
| Main Cash Account Table         | Lists all portfolios and their designated main cash accounts.          | ตารางแสดงรายชื่อพอร์ตโฟลิโอทั้งหมดและบัญชีเงินสดหลักที่ผูกไว้       |
| Preview Button                  | View the detailed link between the portfolio and its cash account.     | ปุ่มสำหรับเรียกดูรายละเอียดการเชื่อมโยงของพอร์ตและบัญชีเงินสด       |
| Edit Button                     | Change or update the designated main cash account for a portfolio.     | ปุ่มสำหรับแก้ไขหรือเปลี่ยนบัญชีเงินสดหลักที่เลือกไว้                |
| Delete Button                   | Remove the main cash account link from the system.                     | ปุ่มสำหรับลบการเชื่อมโยงบัญชีเงินสดหลักออกจากระบบ                   |
| Pending Main Cash Account Table | Account links awaiting authorized approval before being active.        | ตารางแสดงรายการเชื่อมโยงบัญชีเงินสดหลักที่รอการอนุมัติ              |



### Create Main Cash Account

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (40).png" alt=""><figcaption></figcaption></figure></div>

| **Component**     | **Description (English)**                                       | **คำอธิบาย (ภาษาไทย)**                                      |
| ----------------- | --------------------------------------------------------------- | ----------------------------------------------------------- |
| Portfolio         | Select the specific portfolio to link with a main cash account. | เลือกพอร์ตโฟลิโอที่ต้องการกำหนดบัญชีเงินสดหลัก              |
| Main Cash Account | Choose the Cash Security to be used as the default account.     | เลือกหลักทรัพย์เงินสด (Cash Security) ที่จะใช้เป็นบัญชีหลัก |
| Create Button     | Save the association and submit it for authorized approval.     | ปุ่มสำหรับบันทึกการเชื่อมโยงข้อมูลและส่งไปรอการอนุมัติ      |
| Cancel Button     | Discard the current selection and return to the main list.      | ปุ่มสำหรับยกเลิกสิ่งที่เลือกไว้และกลับสู่หน้ารายการหลัก     |



### Don't forget to approve the action in the Pending Table to finalize the setup of your main cash account.
