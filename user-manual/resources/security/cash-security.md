# Cash Security

### Cash Security

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (36).png" alt=""><figcaption></figcaption></figure></div>

| **Component**               | **Description (English)**                                          | **คำอธิบาย (ภาษาไทย)**                                         |
| --------------------------- | ------------------------------------------------------------------ | -------------------------------------------------------------- |
| Cash Security Code (Search) | Search for a specific cash security or bank account record.        | ช่องสำหรับค้นหาหลักทรัพย์ประเภทเงินสดด้วยรหัสที่ระบุ           |
| Cash Security Preset        | Access predefined templates for quick cash security setup.         | เมนูสำหรับเลือกใช้เทมเพลตมาตรฐานในการตั้งค่าเงินสด             |
| Create Cash Security        | Open the setup to register a new cash or bank-related security.    | ปุ่มสำหรับเริ่มสร้างและลงทะเบียนหลักทรัพย์ประเภทเงินสดใหม่     |
| Cash Security Table         | Lists all registered cash securities currently in the system.      | ตารางแสดงรายชื่อหลักทรัพย์ประเภทเงินสดทั้งหมดที่บันทึกไว้      |
| Preview Button              | View the full properties, linked bank, and currency details.       | ปุ่มสำหรับเรียกดูรายละเอียดและการตั้งค่าทั้งหมดของเงินสดนั้น ๆ |
| Edit Button                 | Modify the attributes or linking information of the cash security. | ปุ่มสำหรับแก้ไขข้อมูลหรือการเชื่อมโยงของหลักทรัพย์เงินสดเดิม   |
| Delete Button               | Remove the cash security record from the system.                   | ปุ่มสำหรับลบข้อมูลหลักทรัพย์ประเภทเงินสดออกจากระบบ             |
| Pending Cash Security Table | Cash securities awaiting authorized approval before active use.    | ตารางแสดงรายการเงินสดที่รอการอนุมัติก่อนเริ่มใช้งานจริง        |



### Create Cash Security

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (37).png" alt=""><figcaption></figcaption></figure></div>

| **Component**        | **Description (English)**                                               | **คำอธิบาย (ภาษาไทย)**                                    |
| -------------------- | ----------------------------------------------------------------------- | --------------------------------------------------------- |
| From Preset          | Select a pre-configured template to auto-fill cash security fields.     | เลือกพรีเซ็ตที่ตั้งค่าไว้ล่วงหน้าเพื่อกรอกข้อมูลอัตโนมัติ |
| Security Code        | A unique identification code for this cash security (e.g., SAVING-SCB). | รหัสเฉพาะสำหรับระบุหลักทรัพย์เงินสดนี้                    |
| Description          | A brief summary or name of the cash security.                           | คำอธิบายสั้นๆ หรือชื่อของหลักทรัพย์เงินสด                 |
| Portfolio            | Select the specific portfolio this cash security belongs to.            | เลือกพอร์ตโฟลิโอที่หลักทรัพย์เงินสดนี้สังกัดอยู่          |
| Bank Account No      | The associated bank account number from the system's records.           | หมายเลขบัญชีธนาคารที่ผูกกับหลักทรัพย์นี้                  |
| Security Type        | Categorization of the security (e.g., Cash, Saving Account).            | ประเภทของหลักทรัพย์ (เช่น เงินสด, บัญชีออมทรัพย์)         |
| Business Sector      | Industry classification (e.g., Banking, Finance).                       | การจัดกลุ่มตามประเภทธุรกิจ                                |
| Exchange             | The trading exchange or marketplace reference.                          | ตลาดหลักทรัพย์หรือแหล่งอ้างอิงการแลกเปลี่ยน               |
| Remark               | Additional notes or context regarding this security.                    | หมายเหตุเพิ่มเติมหรือข้อมูลประกอบอื่น ๆ                   |
| Int. Rate Condition  | Link to the applicable interest-rate rule or formula.                   | เลือกเงื่อนไขอัตราดอกเบี้ยที่ต้องการนำมาปรับใช้           |
| Spread               | The percentage added to the base interest rate (if any).                | ส่วนต่างอัตราดอกเบี้ยที่บวกเพิ่มจากอัตราอ้างอิง           |
| Coupon Frequency     | The frequency at which interest is paid (e.g., Monthly, Semi-Annual).   | ความถี่ในการจ่ายดอกเบี้ย (เช่น รายเดือน, ทุก 6 เดือน)     |
| Interest Calc. Basis | The method used to calculate interest (Day-count convention).           | วิธีการคำนวณดอกเบี้ย (เกณฑ์การนับวัน)                     |
| Create Button        | Save the configuration and register the cash security.                  | ปุ่มสำหรับบันทึกข้อมูลและลงทะเบียนหลักทรัพย์เงินสด        |
| Cancel Button        | Discard current input and return to the main list.                      | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้และกลับสู่หน้าก่อนหน้า      |
