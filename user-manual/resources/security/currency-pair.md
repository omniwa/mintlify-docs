# Currency Pair

### Currency Pair

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (28).png" alt=""><figcaption></figcaption></figure></div>

| **Component**               | **Description (English)**                                                 | **คำอธิบาย (ภาษาไทย)**                                             |
| --------------------------- | ------------------------------------------------------------------------- | ------------------------------------------------------------------ |
| Currency Pair Code (Search) | Search for a specific pair using its combined code (e.g., USD/THB).       | ช่องสำหรับค้นหาคู่สกุลเงินด้วยรหัสที่ระบุ (เช่น USD/THB)           |
| Create Currency Pair        | Open the setup to define and add a new currency pair to the system.       | ปุ่มสำหรับเริ่มตั้งค่าและเพิ่มคู่สกุลเงินใหม่เข้าสู่ระบบ           |
| Currency Pair Table         | Lists all registered currency pairs used for exchange rate tracking.      | ตารางแสดงรายชื่อคู่สกุลเงินทั้งหมดที่ใช้ในการติดตามอัตราแลกเปลี่ยน |
| Preview Button              | View the detailed configuration and historical rate settings.             | ปุ่มสำหรับเรียกดูการตั้งค่าและรายละเอียดของคู่สกุลเงินนั้น ๆ       |
| Edit Button                 | Modify the attributes or scaling of an existing currency pair.            | ปุ่มสำหรับแก้ไขคุณลักษณะหรือการกำหนดค่าของคู่สกุลเงินเดิม          |
| Delete Button               | Remove the currency pair record from the system.                          | ปุ่มสำหรับลบข้อมูลคู่สกุลเงินออกจากระบบ                            |
| Pending Currency Pair Table | Currency pairs recently created or modified awaiting authorized approval. | ตารางแสดงคู่สกุลเงินที่รอการอนุมัติก่อนเริ่มใช้งานจริงในระบบ       |



### Create Currency Pair

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (29).png" alt=""><figcaption></figcaption></figure></div>

| **Component**      | **Description (English)**                                              | **คำอธิบาย (ภาษาไทย)**                                         |
| ------------------ | ---------------------------------------------------------------------- | -------------------------------------------------------------- |
| Currency Pair Code | A unique identifier for the pair (e.g., USD/THB, EUR/USD).             | รหัสเฉพาะสำหรับระบุคู่สกุลเงิน (เช่น USD/THB)                  |
| Commodity          | Toggle to indicate if the pair involves a commodity (e.g., Gold, Oil). | ระบุว่าคู่สกุลเงินนี้เกี่ยวข้องกับสินค้าโภคภัณฑ์หรือไม่        |
| Quoted             | Set the reference logic for exchange-rate display and calculation.     | กำหนดรูปแบบการแสดงราคาอ้างอิงเพื่อใช้ในการคำนวณอัตราแลกเปลี่ยน |
| Create Button      | Save the pair configuration and submit it for approval.                | ปุ่มสำหรับบันทึกการตั้งค่าคู่สกุลเงินและส่งไปรอการอนุมัติ      |
| Cancel Button      | Discard the current input and return to the main list.                 | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้และกลับสู่หน้ารายการหลัก         |





### Don't forget to approve the action in the Pending Table to finalize the setup of your new currency pair.
