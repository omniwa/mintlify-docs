# Calendar

### Calendar

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (41).png" alt=""><figcaption></figcaption></figure></div>

| **Component**          | **Description (English)**                                                          | **คำอธิบาย (ภาษาไทย)**                                        |
| ---------------------- | ---------------------------------------------------------------------------------- | ------------------------------------------------------------- |
| Calendar Code (Search) | Search for a specific financial calendar by its unique code.                       | ช่องสำหรับค้นหาปฏิทินด้วยรหัส (เช่น TH\_Bank, US\_Market)     |
| Create Calendar        | Open the setup to define a new holiday calendar for a specific market or currency. | ปุ่มสำหรับเริ่มสร้างปฏิทินวันหยุดใหม่เข้าสู่ระบบ              |
| Calendar Table         | Lists all registered calendars and a summary of their configured holidays.         | ตารางแสดงรายชื่อปฏิทินทั้งหมดและข้อมูลสรุปวันหยุดที่บันทึกไว้ |
| Preview Button         | View the list of specific holiday dates within the selected calendar.              | ปุ่มสำหรับเรียกดูรายการวันหยุดทั้งหมดในปฏิทินนั้น ๆ           |
| Edit Button            | Add or remove holiday dates and update calendar descriptions.                      | ปุ่มสำหรับแก้ไขหรือเพิ่ม/ลดวันหยุดในปฏิทินเดิม                |
| Delete Button          | Remove the calendar record from the system database.                               | ปุ่มสำหรับลบข้อมูลปฏิทินออกจากระบบ                            |
| Pending Calendar Table | Calendars or holiday updates awaiting authorized approval.                         | ตารางแสดงรายชื่อปฏิทินที่รอการอนุมัติก่อนเริ่มใช้งานจริง      |



### Create Calendar

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (42).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                                       | **คำอธิบาย (ภาษาไทย)**                               |
| --------------------- | --------------------------------------------------------------- | ---------------------------------------------------- |
| Code                  | A unique identifier for the calendar (e.g., TH\_SET, US\_NYSE). | รหัสเฉพาะสำหรับระบุปฏิทิน (เช่น TH\_SET)             |
| Description           | A brief summary of the calendar's purpose or region.            | คำอธิบายสั้นๆ เกี่ยวกับปฏิทินหรือขอบเขตการใช้งาน     |
| Working Days / Week   | Define the standard number of business days in a week.          | กำหนดจำนวนวันทำงานปกติในหนึ่งสัปดาห์                 |
| Date                  | The specific date designated as a holiday.                      | วันที่ที่กำหนดให้เป็นวันหยุดในระบบ                   |
| Name                  | The official title of the holiday (e.g., New Year's Day).       | ชื่อเรียกของวันหยุด (เช่น วันขึ้นปีใหม่)             |
| Description (Holiday) | Detailed notes regarding the specific holiday entry.            | รายละเอียดหรือหมายเหตุเพิ่มเติมเกี่ยวกับวันหยุดนั้นๆ |
| Delete Button         | Remove a specific holiday row from the calendar list.           | ปุ่มสำหรับลบแถววันหยุดที่ไม่ต้องการออกจากรายการ      |
| Add Holiday           | Append a new row to include an additional holiday date.         | ปุ่มสำหรับเพิ่มแถวใหม่เพื่อระบุวันหยุดเพิ่มเติม      |
| Create Button         | Save the calendar configuration and submit for approval.        | ปุ่มสำหรับบันทึกปฏิทินและส่งไปรอการอนุมัติ           |
| Cancel Button         | Discard the current setup and return to the management page.    | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้และกลับสู่หน้าหลัก     |



### Don't forget to approve the action in the Pending Table to finalize the creation or update of your financial calendar.
