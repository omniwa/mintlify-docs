### Mapping Investment Type

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (52).png" alt=""><figcaption></figcaption></figure></div>

| **Component**                         | **Description (English)**                                           | **คำอธิบาย (ภาษาไทย)**                                      |
| ------------------------------------- | ------------------------------------------------------------------- | ----------------------------------------------------------- |
| Mapping Investment Type Code (Search) | Search for specific investment type mapping records using the code. | ช่องสำหรับค้นหาข้อมูลการจับคู่ประเภทการลงทุนด้วยรหัสที่ระบุ |
| Create Mapping Investment Type        | Open the setup to define a new mapping between investment types.    | ปุ่มสำหรับเริ่มสร้างการจับคู่ประเภทการลงทุนใหม่เข้าสู่ระบบ  |

| Preview Button                        | View the detailed logic and linked categories of the mapping.       | ปุ่มสำหรับเรียกดูรายละเอียดและการเชื่อมโยงของข้อมูลนั้น ๆ   |
| Edit Button                           | Modify the attributes or linked investment categories.              | ปุ่มสำหรับแก้ไขหรือปรับปรุงข้อมูลการจับคู่เดิม              |
| Delete Button                         | Remove the investment type mapping record from the system.          | ปุ่มสำหรับลบข้อมูลการจับคู่ประเภทการลงทุนออกจากระบบ         |

{% tab title="Create/Edit" %}
### Create/Edit Mapping Investment Type

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (53).png" alt=""><figcaption></figcaption></figure></div>

| **Component**      | **Description (English)**                                               | **คำอธิบาย (ภาษาไทย)**                                      |
| ------------------ | ----------------------------------------------------------------------- | ----------------------------------------------------------- |
| Mapping Code       | A unique identifier for this specific investment mapping.               | รหัสเฉพาะสำหรับระบุการตั้งค่าการจับคู่ประเภทการลงทุนนี้     |
| Asset / Liab Code  | Classification code to identify if this type is an Asset or Liability.  | รหัสระบุประเภทเพื่อแยกแยะว่าเป็นหมวดสินทรัพย์หรือหนี้สิน    |
| Description        | A brief summary or name of the investment type category.                | คำอธิบายสั้นๆ หรือชื่อของหมวดหมู่การลงทุน                   |
| Description2       | Secondary or additional details for more granular identification.       | คำอธิบายเพิ่มเติมเพื่อใช้ระบุรายละเอียดที่เจาะจงมากขึ้น     |
| Security Available | Lists all registered securities that have not been assigned to a group. | รายชื่อหลักทรัพย์ทั้งหมดในระบบที่ยังไม่ได้ถูกจัดกลุ่ม       |
| Move Button (>)    | Transfer the selected securities from Available to the Selected list.   | ปุ่มสำหรับย้ายหลักทรัพย์ที่เลือกไปยังรายการที่ต้องการจับคู่ |
| Security Selected  | Securities that are successfully mapped to this investment type.        | รายชื่อหลักทรัพย์ที่ถูกเลือกเพื่อจัดอยู่ในหมวดหมู่นี้       |
| Create Button      | Save the mapping configuration and register it in the system.           | ปุ่มสำหรับบันทึกการตั้งค่าการจับคู่เข้าสู่ระบบ              |
| Cancel Button      | Discard current entries and return to the management table.             | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้และกลับสู่หน้าหลัก            |
