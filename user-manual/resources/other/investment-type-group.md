# Investment Type Group

### Investment Type Group

<figure><img src="../../.gitbook/assets/image (314).png" alt=""><figcaption></figcaption></figure>

| **Component**                       | **Description (English)**                                                  | **คำอธิบาย (ภาษาไทย)**                                       |
| ----------------------------------- | -------------------------------------------------------------------------- | ------------------------------------------------------------ |
| Investment Type Group Code (Search) | Search for a specific investment group using its unique code.              | ช่องสำหรับค้นหากลุ่มประเภทการลงทุนด้วยรหัสที่ระบุ            |
| Create Investment Type Group        | Open the interface to define a new overarching investment group.           | ปุ่มสำหรับเริ่มสร้างกลุ่มประเภทการลงทุนใหม่เข้าสู่ระบบ       |
| Investment Type Group Table         | Lists all registered groups, providing a high-level view of asset classes. | ตารางแสดงรายชื่อกลุ่มประเภทการลงทุนทั้งหมดที่บันทึกไว้       |
| Group Desc1                         | The primary description of the investment group (often in English).        | คำอธิบายหลักของกลุ่ม (มักใช้ชื่อที่เป็นทางการหรือภาษาอังกฤษ) |
| Group Desc2                         | The secondary or local description for the group (often in Thai).          | คำอธิบายเพิ่มเติมหรือชื่อกลุ่มภาษาไทยเพื่อความชัดเจน         |
| Preview Button                      | View the full configuration and sub-types within the group.                | ปุ่มสำหรับเรียกดูรายละเอียดและการตั้งค่าภายในกลุ่มนั้น ๆ     |
| Edit Button                         | Modify the group codes, descriptions, or linked investment types.          | ปุ่มสำหรับแก้ไขหรือปรับปรุงข้อมูลของกลุ่มเดิม                |
| Delete Button                       | Remove the investment type group record from the system.                   | ปุ่มสำหรับลบข้อมูลกลุ่มประเภทการลงทุนออกจากระบบ              |



### Create Investment Type Group

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (315).png" alt=""><figcaption></figcaption></figure></div>

| **Component** | **Description (English)**                                            | **คำอธิบาย (ภาษาไทย)**                                 |
| ------------- | -------------------------------------------------------------------- | ------------------------------------------------------ |
| Group Code    | A unique identifier for the investment group (e.g., EQUITY\_GLOBAL). | รหัสเฉพาะสำหรับระบุกลุ่มประเภทการลงทุน                 |
| Group Desc    | The primary title or description of the investment group.            | คำอธิบายหลักหรือชื่อเรียกของกลุ่มการลงทุน              |
| Group Desc2   | Secondary or additional clarification for the group.                 | คำอธิบายเพิ่มเติมหรือชื่อกลุ่มภาษาไทยเพื่อความชัดเจน   |
| Create Button | Save the group configuration and register it in the system.          | ปุ่มสำหรับบันทึกการตั้งค่ากลุ่มและลงทะเบียนเข้าสู่ระบบ |
| Cancel Button | Discard current input and return to the management list.             | ปุ่มสำหรับยกเลิกสิ่งที่กรอกไว้และกลับสู่หน้ารายการหลัก |
