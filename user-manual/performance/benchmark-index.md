# Benchmark Index

### Benchmark Index

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (122).png" alt=""><figcaption></figcaption></figure></div>

| **Component**           | **Description (English)**                              | **คำอธิบาย (ภาษาไทย)**                                            |
| ----------------------- | ------------------------------------------------------ | ----------------------------------------------------------------- |
| Benchmark Indexes Table | Lists every benchmark configured in the system.        | ตารางแสดงรายการดัชนีชี้วัด (Benchmark) ทั้งหมดที่ตั้งค่าไว้ในระบบ |
| + Add New Row Button    | Adds a new benchmark row to the table.                 | ปุ่มสำหรับเพิ่มแถวเพื่อสร้างดัชนีชี้วัดรายการใหม่ลงในตาราง        |
| Name                    | Name of the benchmark.                                 | ชื่อของดัชนีชี้วัด                                                |
| Type                    | Category of the benchmark.                             | ประเภทหรือหมวดหมู่ของดัชนีชี้วัด                                  |
| Start at                | Date the benchmark begins to track values.             | วันที่เริ่มต้นการติดตามค่าของดัชนีชี้วัด                          |
| Auto Fetch At           | Date on which the system automatically pulls new data. | วันที่ระบบจะเริ่มดึงข้อมูลใหม่โดยอัตโนมัติสำหรับดัชนีนั้นๆ        |
| Value Day               | Benchmark value for the displayed day.                 | ค่าของดัชนีชี้วัด ณ วันที่แสดงผล                                  |
| Rate                    | Percentage return or fixed rate.                       | อัตราผลตอบแทนเป็นเปอร์เซ็นต์ หรืออัตราคงที่                       |
| Edit Button             | Edit the Benchmark Index.                              | ปุ่มสำหรับแก้ไขข้อมูลการตั้งค่าของดัชนีชี้วัด                     |
| Delete Button           | Delete the Benchmark Index.                            | ปุ่มสำหรับลบรายการดัชนีชี้วัดออกจากระบบ                           |
