# Performance Policy

### Performance Policy

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (167).png" alt=""><figcaption></figcaption></figure></div>

| **Component**            | **Description (English)**                           | **คำอธิบาย (ภาษาไทย)**                                   |
| ------------------------ | --------------------------------------------------- | -------------------------------------------------------- |
| Performance Policy Table | Lists available performance policies in the system. | ตารางแสดงรายการนโยบายการวัดผลการดำเนินงานที่มีอยู่ในระบบ |
| + Add New Policy Button  | Create a new performance policy.                    | ปุ่มสำหรับสร้างนโยบายการวัดผลการดำเนินงานรายการใหม่      |
| 👁 View Button           | Open the policy details for review.                 | ปุ่มสำหรับเรียกดูรายละเอียดการตั้งค่าของนโยบาย           |
| 🖊 Edit Button           | Modify the selected performance policy.             | ปุ่มสำหรับแก้ไขข้อมูลนโยบายการวัดผล                      |
| 🗑️ Delete Button        | Delete the selected performance policy.             | ปุ่มสำหรับลบรายการนโยบายออกจากระบบ                       |



### **Create a Performance Policy**

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (168).png" alt=""><figcaption></figcaption></figure></div>

| **Component** | **Description (English)**                       | **คำอธิบาย (ภาษาไทย)**                           |
| ------------- | ----------------------------------------------- | ------------------------------------------------ |
| Name          | Name of the performance policy.                 | ชื่อของนโยบายการวัดผลการดำเนินงาน                |
| Cancel Button | Cancel the policy creation and discard changes. | ปุ่มสำหรับยกเลิกการสร้างนโยบายและไม่บันทึกข้อมูล |
| Create Button | Save the new performance policy to the system.  | ปุ่มสำหรับบันทึกนโยบายการวัดผลรายการใหม่ลงในระบบ |



### **Edit a Performance Policy**

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (169).png" alt=""><figcaption></figcaption></figure></div>

| **Component**           | **Description (English)**                          | **คำอธิบาย (ภาษาไทย)**                                    |
| ----------------------- | -------------------------------------------------- | --------------------------------------------------------- |
| Name                    | Name of the performance policy.                    | ชื่อของนโยบายการวัดผลการดำเนินงาน                         |
| Members                 | Portfolios that belong to this policy.             | รายชื่อพอร์ตโฟลิโอที่อยู่ภายใต้นโยบายการวัดผลนี้          |
| Join Date               | Date the portfolio was added to the policy.        | วันที่พอร์ตโฟลิโอเริ่มเข้าร่วมใช้นโยบายการวัดผลนี้        |
| Leave Date              | Date the portfolio left the policy.                | วันที่พอร์ตโฟลิโอสิ้นสุดการใช้นโยบายการวัดผลนี้           |
| Edit Button             | Modify member portfolio details.                   | ปุ่มสำหรับแก้ไขรายละเอียดข้อมูลสมาชิกในนโยบาย             |
| Delete Button           | Remove the portfolio from the policy.              | ปุ่มสำหรับลบพอร์ตโฟลิโอออกจากนโยบายการวัดผล               |
| + Add New Member Button | Add another portfolio to the policy.               | ปุ่มสำหรับเพิ่มพอร์ตโฟลิโอใหม่เข้ามาเป็นสมาชิกในนโยบาย    |
| Cancel Button           | Discard changes and return to the previous screen. | ปุ่มสำหรับยกเลิกการเปลี่ยนแปลงทั้งหมดและไม่บันทึกข้อมูล   |
| Save Button             | Save changes to the policy member list.            | ปุ่มสำหรับบันทึกการเปลี่ยนแปลงข้อมูลสมาชิกทั้งหมดลงในระบบ |
