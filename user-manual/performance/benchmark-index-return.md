# Benchmark Index Return

### Benchmark Index Return

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (160).png" alt=""><figcaption></figcaption></figure></div>

| **Component**                | **Description (English)**                                    | **คำอธิบาย (ภาษาไทย)**                                                  |
| ---------------------------- | ------------------------------------------------------------ | ----------------------------------------------------------------------- |
| Select Date                  | Date range used to retrieve benchmark data.                  | ช่วงวันที่ที่ใช้ในการดึงข้อมูลดัชนีชี้วัดมาแสดงผล                       |
| Benchmark Name               | The name of the benchmark currently being viewed.            | ชื่อของดัชนีชี้วัดที่กำลังแสดงข้อมูลอยู่ในขณะนี้                        |
| Benchmark Table              | Lists each entry including: date, value, and adjusted value. | ตารางแสดงรายการข้อมูล ประกอบด้วย วันที่, ค่าดัชนี และค่าที่ปรับปรุงแล้ว |
| Export Button                | Export the benchmark data into an external file.             | ปุ่มสำหรับส่งออกข้อมูลดัชนีชี้วัดเป็นไฟล์ภายนอก                         |
| Expand / Collapse All Button | Show or hide all data sections at once.                      | ปุ่มสำหรับแสดงหรือซ่อนรายละเอียดทุกส่วนพร้อมกัน                         |
| + Add Row Button             | Add a new benchmark record manually.                         | ปุ่มสำหรับเพิ่มแถวเพื่อกรอกข้อมูลดัชนีชี้วัดรายการใหม่                  |
| Select Benchmark             | Choose which benchmark the new row applies to.               | ส่วนสำหรับเลือกดัชนีชี้วัดที่ต้องการบันทึกข้อมูลในแถวใหม่               |
| Date                         | Date of the specific benchmark entry.                        | วันที่ของรายการข้อมูลดัชนีชี้วัดนั้นๆ                                   |
| Value                        | Recorded benchmark value for that entry.                     | ค่าของดัชนีชี้วัดที่บันทึกไว้                                           |
| Adjusted Value               | The benchmark value after any necessary adjustments.         | ค่าของดัชนีชี้วัดหลังจากทำการปรับปรุงข้อมูลแล้ว                         |



#### Create new Benchmark data

<figure><img src="../.gitbook/assets/image (161).png" alt=""><figcaption></figcaption></figure>

| **Component** | **Description (English)**                            | **คำอธิบาย (ภาษาไทย)**                              |
| ------------- | ---------------------------------------------------- | --------------------------------------------------- |
| Create Button | Save the new benchmark entry to the system.          | ปุ่มสำหรับบันทึกข้อมูลดัชนีชี้วัดรายการใหม่ลงในระบบ |
| Cancel Button | Discard the entry and return to the previous screen. | ปุ่มสำหรับยกเลิกการกรอกข้อมูลและไม่บันทึกรายการ     |
