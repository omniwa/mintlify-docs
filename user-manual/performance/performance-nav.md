  deduction. It lets investors see the fund’s valuation in its purest form,
  before year-end fee adjustments are applied.

# Performance Nav

### Performance NAV

<figure><img src="../.gitbook/assets/image (163).png" alt=""><figcaption></figcaption></figure>

| **Component**                | **Description (English)**                         | **คำอธิบาย (ภาษาไทย)**                                                                 |
| ---------------------------- | ------------------------------------------------- | -------------------------------------------------------------------------------------- |
| Dates                        | Date range for the NAV view.                      | ช่วงวันที่ที่ต้องการเรียกดูข้อมูลมูลค่าทรัพย์สินสุทธิ (NAV)                            |
| Portfolio Group              | Group of portfolios being analysed.               | กลุ่มของพอร์ตโฟลิโอที่ต้องการนำมาวิเคราะห์                                             |
| Search Button                | Run the search based on selected filters.         | ปุ่มสำหรับกดเพื่อเริ่มค้นหาข้อมูลตามเงื่อนไขที่กำหนด                                   |
| Performance NAV Table        | NAV values per portfolio, including NAV per unit. | ตารางแสดงค่า NAV ของแต่ละพอร์ตโฟลิโอ รวมถึงมูลค่าทรัพย์สินสุทธิต่อหน่วย (NAV per unit) |
| Export Button                | Download an Excel file for further analysis.      | ปุ่มสำหรับดาวน์โหลดข้อมูลในรูปแบบไฟล์ Excel เพื่อนำไปวิเคราะห์ต่อ                      |
| Expand / Collapse All Button | Show or hide all sections at once.                | ปุ่มสำหรับกางออกหรือยุบรายละเอียดทุกส่วนพร้อมกัน                                       |
