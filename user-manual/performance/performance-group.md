# Performance Group

### Performance Group

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (165).png" alt=""><figcaption></figcaption></figure></div>

| **Component**                   | **Description (English)**                         | **คำอธิบาย (ภาษาไทย)**                                                 |
| ------------------------------- | ------------------------------------------------- | ---------------------------------------------------------------------- |
| Performance Group Table         | Lists portfolios that have been grouped together. | ตารางแสดงรายชื่อพอร์ตโฟลิโอที่ถูกจัดกลุ่มไว้ด้วยกัน                    |
| Create Performance Group Button | Open the form to create a new performance group.  | ปุ่มสำหรับเปิดแบบฟอร์มเพื่อสร้างกลุ่มการวัดผล (Performance Group) ใหม่ |



### **Create a Performance Group**

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (166).png" alt=""><figcaption></figcaption></figure></div>

| **Component**       | **Description (English)**                             | **คำอธิบาย (ภาษาไทย)**                                   |
| ------------------- | ----------------------------------------------------- | -------------------------------------------------------- |
| Name                | Name of the performance group.                        | ชื่อของกลุ่มการวัดผล (Performance Group)                 |
| Portfolio Available | List of portfolios that can be added to the group.    | รายชื่อพอร์ตโฟลิโอทั้งหมดที่สามารถเลือกเพิ่มเข้ากลุ่มได้ |
| Button >            | Move the highlighted portfolios to the Selected list. | ปุ่มสำหรับย้ายพอร์ตโฟลิโอที่เลือกไปยังรายการ "Selected"  |
| Portfolio Selected  | Portfolios that have been assigned to the group.      | รายชื่อพอร์ตโฟลิโอที่ถูกเลือกให้อยู่ในกลุ่มนี้           |
| Create              | Save the new performance group to the system.         | ปุ่มสำหรับบันทึกการสร้างกลุ่มใหม่ลงในระบบ                |
