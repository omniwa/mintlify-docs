# Login

### Login

Open your web browser and go to [https://demo.level11.dev/login](https://demo.level11.dev/login) to access the Login page. Every user must sign in before any other module can be used.

<div align="center" data-with-frame="true"><figure><img src="../.gitbook/assets/image (125).png" alt=""><figcaption></figcaption></figure></div>

### Step to Login

{% stepper %}
{% step %}
#### Username

Enter the username assigned to you by the administrator.
{% endstep %}

{% step %}
#### Password

Enter the password that matches the username.
{% endstep %}

{% step %}
#### Login button

Click to sign in. If the credentials are correct, the application opens on the Home page.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
_If sign-in fails, double-check the username and password. Ask the IT/Admin team to reset your password._
{% endhint %}
