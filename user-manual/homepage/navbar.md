# Navbar

### Navbar

After logging in successfully, the Navbar appears on every page. It is the primary way to navigate between modules and to manage your session.



<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (150).png" alt=""><figcaption></figcaption></figure></div>



### **Navbar Elements**

| **Component** | **Description (English)**                                                | **คำอธิบาย (ภาษาไทย)**                                                  |
| ------------- | ------------------------------------------------------------------------ | ----------------------------------------------------------------------- |
| Home Logo     | Returns you to the main landing page from anywhere in the application.   | ปุ่มโลโก้สำหรับกลับไปยังหน้าหลัก (Dashboard) ได้จากทุกหน้าในโปรแกรม     |
| Name          | Displays the name of the user who is currently signed in.                | แสดงชื่อของผู้ใช้งานที่กำลังล็อกอินเข้าสู่ระบบในขณะนั้น                 |
| Role          | Shows the user’s role, which determines available menus and permissions. | แสดงสิทธิ์การใช้งานหรือตำแหน่ง ซึ่งจะเป็นตัวกำหนดการเข้าถึงเมนูต่าง ๆ   |
| Language      | Switches the interface language (e.g., English / Thai).                  | ปุ่มสำหรับสลับเปลี่ยนภาษาในการแสดงผลของหน้าจอโปรแกรม                    |
| Manual Lock   | Click the lock icon to secure your screen without logging out.           | ปุ่มรูปล็อกสำหรับล็อกหน้าจอชั่วคราวเพื่อความปลอดภัยโดยไม่ต้องออกจากระบบ |
| Log Out       | Signs out of the system and returns to the Login screen.                 | ปุ่มสำหรับออกจากระบบเพื่อกลับไปยังหน้าเข้าใช้งานหลัก                    |



{% hint style="info" %}
_Tip: The menus shown in the Navbar depend on the role assigned to your user account. If a menu is missing, contact your administrator to confirm permissions._
{% endhint %}

{% hint style="info" %}
_Tip: To verify the current software build, users can hover their mouse over the Logo at the top of the sidebar._&#x20;
{% endhint %}
