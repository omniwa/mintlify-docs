# Close Transaction

The FX module manages every foreign exchange transaction needed to support international investments — from creating an FX trade to posting it, closing it, and viewing the live FX positions of each portfolio.

### Close Transaction

| **Component**          | **Description (English)**                              | **คำอธิบาย (ภาษาไทย)**                                                          |
| ---------------------- | ------------------------------------------------------ | ------------------------------------------------------------------------------- |
| Transaction Trade Date | Trading date filter for transactions.                  | ตัวกรองวันที่ทำรายการซื้อขายเพื่อค้นหาธุรกรรมที่ต้องการ                         |
| Date                   | Trade execution date selector.                         | ส่วนสำหรับเลือกวันที่ที่มีการทำรายการจริง                                       |
| Columns                | Adjust and customize the visible columns in the table. | ปรับแต่งและเลือกคอลัมน์ที่ต้องการให้แสดงผลในตาราง                               |
| Grouping               | Group FX transactions into categories.                 | จัดกลุ่มรายการธุรกรรม FX ตามหมวดหมู่เพื่อให้ง่ายต่อการตรวจสอบ                   |
| Orders Table           | Lists confirmed and posted FX transactions.            | ตารางแสดงรายการธุรกรรม FX ที่ได้รับการยืนยันและบันทึกบัญชี (Post) เรียบร้อยแล้ว |
| ☒ Unpost Button        | Revert a posted transaction to its previous state.     | ปุ่มสำหรับยกเลิกการบันทึกบัญชี เพื่อย้อนสถานะรายการกลับไปก่อนการ Post           |
| 👁 Preview Button      | Preview the transaction details.                       | ปุ่มสำหรับดูรายละเอียดของธุรกรรมก่อนตัดสินใจทำการ Unpost                        |
