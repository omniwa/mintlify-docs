# Post Transaction

The FX module manages every foreign exchange transaction needed to support international investments — from creating an FX trade to posting it, closing it, and viewing the live FX positions of each portfolio.

### Post Transaction

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (131).png" alt=""><figcaption></figcaption></figure></div>

| **Component**          | **Description (English)**                                  | **คำอธิบาย (ภาษาไทย)**                                             |
| ---------------------- | ---------------------------------------------------------- | ------------------------------------------------------------------ |
| Transaction Trade Date | Trading date filter for transactions.                      | ตัวกรองวันที่ทำรายการซื้อขายเพื่อค้นหาธุรกรรม                      |
| Date                   | Trade execution date selector.                             | ส่วนสำหรับเลือกวันที่ที่มีการทำรายการจริง                          |
| Columns                | Adjust and customize the visible columns in the table.     | ปรับแต่งและเลือกคอลัมน์ที่ต้องการให้แสดงผลในตาราง                  |
| Filters                | Refine the displayed orders based on specific criteria.    | ตัวกรองขั้นสูงเพื่อคัดกรองรายการคำสั่งซื้อขายที่แสดงผล             |
| Grouping               | Group FX transactions into categories (e.g., by Currency). | จัดกลุ่มรายการธุรกรรม FX ตามหมวดหมู่ (เช่น จัดกลุ่มตามสกุลเงิน)    |
| Orders Table           | Lists confirmed transactions waiting to be posted.         | ตารางแสดงรายการธุรกรรมที่ยืนยันแล้วและกำลังรอการบันทึกบัญชี (Post) |
| ☑ Post Button          | Post the selected transaction to the system.               | ปุ่มสำหรับบันทึกรายการธุรกรรมที่เลือกเข้าสู่ระบบหลัก               |
| 👁 Preview Button      | Preview the transaction details before posting.            | ปุ่มสำหรับตรวจสอบรายละเอียดของธุรกรรมก่อนทำการบันทึกจริง           |
