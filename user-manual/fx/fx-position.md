# FX Position

The FX module manages every foreign exchange transaction needed to support international investments — from creating an FX trade to posting it, closing it, and viewing the live FX positions of each portfolio.

### FX Position

FX Position lets fund managers see the cash position of every selected portfolio, broken down by currency. It is the fastest way to see how much foreign-currency cash is available before placing trades.
