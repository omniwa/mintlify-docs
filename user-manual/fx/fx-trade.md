# FX Trade

The FX module manages every foreign exchange transaction needed to support international investments — from creating an FX trade to posting it, closing it, and viewing the live FX positions of each portfolio.

### FX Trade

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (130).png" alt=""><figcaption></figcaption></figure></div>

| **Component**             | **Description (English)**                                    | **คำอธิบาย (ภาษาไทย)**                                                |
| ------------------------- | ------------------------------------------------------------ | --------------------------------------------------------------------- |
| Create / Trade Date       | Toggle between filtering by creation date or trade date.     | สลับประเภทการกรองระหว่างวันที่สร้างรายการ หรือ วันที่ทำรายการจริง     |
| Date                      | Select the specific date or range used by the filter.        | เลือกวันที่ที่ต้องการใช้ในการกรองข้อมูล                               |
| Columns                   | Adjust and customize the visible columns in the table.       | ปรับแต่งและเลือกคอลัมน์ที่ต้องการให้แสดงผลในตาราง                     |
| Filters                   | Refine the displayed data based on specific criteria.        | ตัวกรองเพื่อคัดกรองข้อมูลรายการที่แสดงผลให้ละเอียดขึ้น                |
| Grouping                  | Group orders into categories (e.g., by Status or Currency).  | จัดกลุ่มรายการธุรกรรมตามหมวดหมู่ที่ต้องการ                            |
| Orders Table              | Lists FX transactions that are confirmed but not yet posted. | ตารางแสดงรายการธุรกรรม FX ที่ยืนยันแล้วแต่ยังไม่ได้บันทึกบัญชี (Post) |
| 🖊 Edit Button            | Modify the details of an existing transaction.               | ปุ่มสำหรับแก้ไขรายละเอียดของรายการธุรกรรม                             |
| ☑ Confirm Button          | Confirm and approve the transaction for further processing.  | ปุ่มสำหรับยืนยันและอนุมัติรายการเพื่อให้พร้อมสำหรับการ Post           |
| 🗑️ Delete Button         | Remove the transaction from the system.                      | ปุ่มสำหรับลบรายการธุรกรรมออกจากระบบ                                   |
| ☒ Unconfirm Button        | Revert a confirmed transaction back to draft status.         | ปุ่มสำหรับยกเลิกการยืนยัน เพื่อดึงรายการกลับมาเป็นสถานะร่าง (Draft)   |
| Create Transaction Button | Open the entry form to create a new FX transaction.          | ปุ่มสำหรับเปิดหน้าต่างกรอกข้อมูลเพื่อสร้างรายการธุรกรรม FX ใหม่       |
