# User Management

### User Management

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (170).png" alt=""><figcaption></figcaption></figure></div>

| **Component**         | **Description (English)**                             | **คำอธิบาย (ภาษาไทย)**                                         |
| --------------------- | ----------------------------------------------------- | -------------------------------------------------------------- |
| Username Search Field | Filter the user list by entering a specific username. | ช่องสำหรับกรองรายชื่อผู้ใช้งานโดยการระบุชื่อผู้ใช้ (Username)  |
| Role Search Field     | Filter the user list by their assigned system roles.  | ช่องสำหรับกรองรายชื่อผู้ใช้งานตามบทบาท (Role) ที่ได้รับมอบหมาย |
| Add New User Button   | Create and register a new user account in the system. | ปุ่มสำหรับสร้างและลงทะเบียนบัญชีผู้ใช้งานใหม่เข้าสู่ระบบ       |
| Role Details Table    | Lists user roles and their associated information.    | ตารางแสดงรายการบทบาทของผู้ใช้งานและข้อมูลที่เกี่ยวข้อง         |
| Edit Button           | Modify existing user role settings and permissions.   | ปุ่มสำหรับแก้ไขการตั้งค่าบทบาทและสิทธิ์การใช้งานของผู้ใช้งาน   |



### Add New User

From the User Management screen, click Add New User Button to open the Add New User form. Fill in every required detail and save.
